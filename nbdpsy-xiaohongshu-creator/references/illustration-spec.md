# 插图提示词体系（PPT 式轮播 · 固定品牌基底）

> 目标：一条笔记的 6–9 张图，看起来像同一个号、同一个设计师做的。做法 = **锁住风格基底，只换主体画面与文字**。本项目用 **Gemini / GPT 出图**，二者能把中文文字渲染进图，所以提示词把"要显示在图上的中文"直接写进去。

## 1. 固定品牌基底（每页提示词都以它开头，逐字照搬）

一套笔记里有两类页面，**共用同一套调色板与人物卡**，只是构图密度不同：

- **情绪页**（封面 P1 / 末页 PN / 隐喻页）：柔和治愈单主体，留白勾人。
- **信息海报页**（中间的内容页）：满版高密度，核心是**图文双通道**——每条信息配一个能解码的具体场景小图/简笔小人（见第 3 节「内容页核心：图文双通道」）。

**品牌调色板（品牌资产，跨所有笔记复用，HEX 不随意改；要换风格只改这一处，全站统一）**：

| 用途 | 颜色 | HEX |
|------|------|-----|
| 主色调（画面/图标基调） | 雾霾蓝灰 | #A8B5C4 |
| 背景 / 暖调（纸质） | 暖米白 | #E8D8C4 |
| 辅助色 | 鼠尾草绿 | #C9D6CE |
| 强调 / 高亮（**可选**：标题关键词、结论红字点睛） | 赭砖红 | #A34B3A |
| 标题主字（默认用下方深灰蓝；想加粗更醒目可选此深色） | 深钢蓝黑 | #2B3A4A |
| 正文 / 说明 | 深灰蓝 | #5A6B7B |

> **NBDpsy 情绪页基底**（封面/末页/隐喻页用）：柔和扁平矢量插画风（soft flat illustration），莫兰迪低饱和配色——以雾霾蓝灰 #A8B5C4、暖米白 #E8D8C4、鼠尾草绿 #C9D6CE 为主，背景暖白；圆润柔和的形体，细腻颗粒/纸质肌理，柔光、无强烈阴影，整体平静治愈；竖版 3:4 构图，主体居中或偏下，**顶部留约 1/3 空白放标题**。信息海报页的构图另见第 3 节「内容页核心」（满版、图文双通道、不留大空白）。

**固定人物卡（出现人物时统一这一个形象，保证全套是"同一个人"）**：

> 圆脸、齐肩微卷短发的东亚年轻女性，穿燕麦色针织衫，神情温和平静。

> 风格基底与人物卡是**品牌资产**：跨所有笔记复用同一套，HEX 色值不要随意改。要换品牌风格时，只改这一处，全站统一。

## 2. 文字入图规范（用户用 Gemini/GPT，可渲染中文）

- **把要显示的中文用引号写进提示词**，并指定位置与层级（大标题 / 小标题 / 说明）。例：`在图片上方留白处写中文大标题"别再说自己想太多"`。
- **信息密度 = 图文共同表达的信息量，不是字数**：封面页(P1)、末页(PN)保持精简（大标题 + 一句副标题即可）；**中间的内容页要让"图+文一起读"就获取一个完整认知**——每个视觉元素都承载具体信息（叙事不装饰）、抽象点落成具体场景、6–10 个信息点有结构地铺满（详见第 3 节「内容页核心」①）。文字总量约 80–120 字只是参考区间，别为凑字数堆空话，更别用"一个大隐喻 + 几句抽象标签"充数。
- **内容页版式不止"要点列表"一种——优先按内容形态选最能出干货的版式**（版式菜单见第 3 节）：
  - **要点卡（信息卡）**：并列 3–6 个平行要点时用（大标题 + 4–6 条「加粗小标题≤10字 + 一行说明≤28字」）。
  - **信息图（图文一体，密度更高、更有干货感）**：当内容有"结构关系"时，别硬塞成要点列表，改用信息图把关系画出来——**对比图**（X vs Y）、**流程/步骤图**（自救 N 步、成因链）、**象限/坐标图**（两维度分类，如依恋四象限）、**隐喻图**（抽象概念可视化，如"冰山下的情绪"）、**数据图**（占比/统计，如"约 6.8%"用圆环或人形阵列）、**拆解/结构图**（一个概念的几根支柱）。信息图靠"图形承载结构 + 短标注承载信息"，同样的干货比纯文字列表更好懂、更耐看。
- **整页信息多、但每个文字块仍短**：无论要点卡还是信息图，每个文字块（标题/标注/说明）都短、各自独立——这样 Gemini/GPT 渲染中文更准，读者也更好扫。**信息图的文字风险更高**（标注点位多）：**"≤6 字"仅指点位式短标注（如隐喻图/象限轴标）**；对比句、类型名、数据说明等各版式已在第 3 节模板里各自给了字数上限，以模板内数字为准。关键专业名词单独放大；出图后**逐字核对**；某工具长文老渲染出错，就精简标注或拆成两页。
- **N 步 / N 条清单类内容**（如"自救几步"）的取舍：**只需让读者看清步骤顺序/概览 → 用第 3 节③流程图，一页画完**（干货感强、省页数）；**每一步都要展开细节（做什么/为什么/小提示）→ 一步一卡**，每卡 ≥3 个子点把单步讲透。无论哪种，**避免只有 1–2 点的稀页**夹在足量页之间，整本密度才一致。
- **指定字体气质**而非具体商用字体名：如"圆润无衬线体""温柔手写体"，颜色用品牌深色（深灰蓝 #5A6B7B）。
- **易错专业名词的退路**：若某页必须出现 `CPTSD` `ICD-11` 等模型易渲染错的拼写/缩写，要么放大字号单独成行让模型更易写对，要么在该页提示词标注"该缩写如渲染错误请后期排版叠加"，并在"页面文字"里照常给出正确文本供人工兜底。
- 出图后**务必人工核对图上每个字**——错字是 AI 出图最常见翻车点。

## 3. 单页提示词模板

> **房规：宁详勿略，把提示词当"施工图"写。** 内容页的每一条提示词都要把「整体结构 + 每个元素画成什么具体画面 + 所有图上文字逐字」写全——**详细提示词能锁死模型的随机性，让整套 6–9 张风格统一**；松散描述靠模型即兴发挥，套图之间必然漂移。下面模板给的就是这个详写颗粒度，照着填、别偷懒简写。可选设计项（粗黑标题/赭红/纸质框）不一定用，但"结构 + 具体画面 + 文字"这三样必须写透。

**封面页 / 末页（精简版）**：

```
【NBDpsy 风格基底 原样照搬】
画面：【本页主体场景，含固定人物卡（若有人物）+ 一两个治愈系小道具，如马克杯/绿植/毛毯/窗光】。
文字：在上方留白处写中文大标题"〇〇〇〇"（圆润无衬线体，深灰蓝色）；下方写一句副标题"〇〇〇〇"。
要求：画面干净、留白充足，竖版 3:4，不出现品牌基底之外的颜色。
```

### 内容页核心：图文双通道 + 高密度（所有内容版式先满足这条，再套下面具体版式）

内容信息图页要对齐"高密度科普长图"的干货感，**唯一硬要求是「图文双通道」**——密度不是靠堆字，是靠**每条知识点都有自己的画面**。**下面每个版式模板开头的「【NBDpsy 风格基底 原样照搬】」= 品牌调色板 + 下面①（②按需叠加）**。

**① 硬要求（缺了就返工）：图文共同表达足量信息**

> **密度 = 图文结合共同表达的信息量，不是字数。** 一个大隐喻 + 三四个抽象短标签（哪怕配了图标）仍是低密度——因为图在"装饰"、文字在"贴标签"，两者没合起来讲清一件事。反例：`抑郁症＝一场雨` 一朵雨云 + 3 个抽象标签「成组出现 / 至少两周 / 有周期」——雨云不承载信息，标签读完不知道"具体长啥样"。

- **图要"叙事"不要"装饰"**：每个图形元素都描绘一个读者能一眼解码的**具体情境**（一个动作、一个场景、一个对照），让"看图"本身就获取信息；不要用一个抽象大隐喻 + 几个泛标签充数。
- **抽象概念落地成具体场景/例子**：别把"成组出现""有周期"这类抽象结论配个泛图标就完事——把它画成看得见的：`成组出现`→并列画出情绪低落、失眠、没胃口、对爱好发呆等几个**具体症状小人**；`至少两周`→画一条**两周日历**几乎每天被标记；`有周期`→画一条**发作↘缓解↗复发**的情绪起伏曲线。图把抽象讲成具体，信息量才真正上去。
- **每条信息 / 每格 / 每节点都配一个各不相同、贴合含义的手绘小图标或简笔小人**（雾霾蓝灰 #A8B5C4 为主、线条清晰）——**不是装饰性小圆点**。图标承载"是什么场景"、文字承载"说什么"，图文双通道才是密度的来源。
- **信息点要足 + 用结构承载信息**：一页 6–10 个信息点，满版铺陈、分 **2–3 区块**组织；能用对比 / 因果 / 前后 / 分类把点之间的**关系也画出来**（关系本身也是信息，见版式菜单 ②–⑧）——覆盖情绪页"顶部留 1/3 空白"那条。

**② 可选设计工具（按需用、保持品牌调性一致即可，都不强制）**

- **标题风格自由**：**默认沿用品牌柔和标题**（圆润无衬线体、深灰蓝 #5A6B7B）即可；想更有海报冲击力也可以加粗、用深钢蓝黑 #2B3A4A——**但不强制粗黑**。
- **强调赭红 #A34B3A**：想突出标题关键词、或给关键结论做"红字点睛"（=被抛弃 / 危险信号 / 你很安全）时可用；不想用则整页保持柔和三色也行。
- **纸质暖框 / 箭头连接**：想要科普海报的"框感"可加纸质暖米背景 + 暖棕描边框；有因果/先后关系时用箭头 → 串联。都是加分项，不是硬指标。

> 判据：把这一页的图和字**一起读**，能否获得一个"信息完整、对得上自己"的认知？能 = 达标。若只看到一个装饰隐喻 + 几句抽象标签、或一页只有 1–2 条、图标是通用圆点 = 密度不足（返工）。**标题粗不粗、有没有红字，都不影响达标。**

**内容页版式菜单——按内容形态选一种（能画成信息图就别只列要点）**：

> **三条使用铁律**：① **按金标准范例的详细度展开**——下面每个模板里的 `〇〇` 只是占位，套用时必须换成具体内容、把每个元素都写成"具体画面"，别把骨架直接丢给模型（详写才能锁住风格、减少随机性）。**每种版式都有一条实测级完整范例，见 `layout-gallery.md`（17 种全收录），写之前先抄对应范例的结构与颗粒度**。② **一套 6–9 页混用 3–5 种不同版式**，别整本都是对比表或都是要点卡。③ **少格版式（格数 < 6，如象限 4 格、对比表 3 行、身体地图）用"格内二次加密"补密度**——每格内再嵌一个具体情境小图 + 一句子标注，用格内颗粒度补足信息量，别让整页只剩几个大标签。此外，每条提示词的「要求」行固定收尾四件套：**仅品牌调色板、满版、所有中文逐字清晰无错字、图内不得出现任何站外导流（微信/二维码/公众号/电话/加群）**。

**易混对的分界**（选错重灾区，先看这四条）：
- **① vs ⑩**：不含勾选/评分 → ①；让读者打勾 + 给"中 ≥N 条"判定阈值 → ⑩。
- **③ vs ⑨**：读者要去做的行动（自救 4 步）→ ③；不由人操作的自然演变（悲伤五阶段）→ ⑨。
- **② vs ⑭ vs ⑫ vs ⑮**：辨析两个都真实的概念 → ②；纠正错误信念（错的打叉）→ ⑭；对别人说的台词 → ⑫；对自己内心的想法替换 → ⑮。
- **⑧ vs ⑪**：一次性塑造过程（开放单向链）→ ⑧；当下自我强化的闭环 → ⑪。

**固定人物卡出场白名单**：叙事性版式（①③⑧⑪⑫⑬⑭⑮）人物小人是主力；抽象结构版式（④⑥⑦⑯）更适合场景物件与图标，人物慎用——别把人物卡贴成装饰，也别在该有叙事小人的页缺席。

**① 分区图标清单（信息卡）**：内容是并列 / 可分组的要点时用（对齐参考图"典型表现 / 身体信号 / 反应模式"那种分区列表）。

```
【NBDpsy 风格基底 原样照搬】
版式：知识卡——顶部大标题"〇〇〇〇"（沿用品牌标题风即可，可选加粗/关键词赭红高亮）。正文分 2–3 个区块，每块一个【方括号】小标题"〇〇"领起；每块内竖排 2–4 条，每条左侧一个雾霾蓝灰手绘小图标或简笔小人（各条图标各不相同、把该条含义画成一个具体场景，可点缀赭红），右侧加粗小标题"〇〇"+ 一行说明"〇〇〇〇"，关键结论词可标赭红。可选纸质暖框。
```

**② 对比图 / 对比表**：内容是"X 和 Y 有什么不同"时用（如 普通回忆 vs 情绪闪回、抑郁症 vs 抑郁型人格）。

```

版式：对比表——顶部大标题"〇〇 vs 〇〇"（两个对象名各用一色高亮：左 雾霾蓝灰、右 赭红）；下方一个两列表格，左列表头 pill"〇〇"（雾霾蓝灰底）、右列表头 pill"〇〇"（赭红/暖底）；表格 3–4 行、每行讲同一维度，左右两格各是「一个双色手绘小图标 + 一句 ≤10 字短句」左右对照；表格外有细描边、行间细分隔线。
要求：左右每行一一对应、每格都有各自的图标、层级清晰，满版竖版 3:4，仅用品牌调色板；所有中文逐字写进图、字体清晰。
```

**③ 流程 / 步骤图**：内容是"分几步 / 一环扣一环"时用（如 自救 5 步、创伤形成链）。

```
要求：箭头流向清晰、步骤块大小一致、间距均匀、每步有图演绎，满版竖版 3:4，仅用品牌调色板；所有中文逐字写进图、字体清晰。
```

## 4. 一套 6–9 张如何保持风格一致

- **风格基底逐字不变**，每页只改"画面主体"和"文字"两行。
- **第 1 张定稿后作参考图**：Gemini/GPT 支持上传参考图，把定稿的 P1 喂进去让后续页"保持同一画风与人物"，比纯文字复现更稳。
- 主色 HEX 在每页都带上，复制粘贴时不要改。
- 人物在多页出现时，每页都带固定人物卡描述，避免长相漂移。

## 6. 视频参考图提示词（去文字版 · 页对页）

每篇笔记文件末尾追加一节 `## 视频参考图提示词`，为轮播的每一页给出一条"去文字版"提示词。

- **用途**：nbdpsy-text-to-video 图生模式用这套提示词生成"无文字同款图"，当每个镜头的首帧参考图。视频阶段会把旁白烧录成字幕——图内若再有文字，会与字幕重叠打架，所以参考图必须完全无文字。本节只产提示词，图由 nbdpsy-text-to-video 产线按需生成（该产线与出图环节同一套宿主自适应逻辑）。
- **推导规则**：拿该页的绘图提示词，**剔除所有图内文字指令**（「文字：…」「顶部大标题"…"」以及基底里"顶部留出约 1/3 空白用于放标题文字"这类为文字服务的构图指令，可改为"构图完整、留白自然"）；**保留品牌基底（三主色 HEX）、固定人物卡、画面主体与版式氛围**；末尾追加一句**「画面中无任何文字」**。
- **页对页**：P1 对 P1、PN 对 PN，画面主体与轮播版**语义呼应**（要点卡页去文字后结构相仿；信息图页有合理例外，见下）。不同版式的去文字化各有讲究：
  - **要点卡页**：把"文字条目"改述为等量的图形元素（小圆图标竖排等），保持画面结构相仿。
  - **信息图页**（对比/流程/象限/数据/拆解）：**别保留没文字的空图表骨架**（空坐标轴、空箭头没有意义、也不适合当视频首帧）；改为回到这一页的**内核场景或隐喻**——用固定人物卡的一个相关情绪画面、或该概念的隐喻意象（如对比页→一个人站在两条岔路前；流程页→拾级而上的台阶；数据页→人群中一个被柔光标出的身影），保留品牌基底与氛围。
  - **隐喻图页**：本就以画面为主，去掉标注即可，隐喻场景原样保留。
- **格式硬约束**：节内每页用加粗标记 `**P1**`…`**PN**` + 围栏提示词；**绝不用 `### PN` 标题**——`### PN` 是「## 配图轮播」的页数计数契约（count_xhs.py 全文数 `### P*`），在本节复用会让页数翻倍误判。
