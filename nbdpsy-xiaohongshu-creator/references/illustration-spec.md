# 插图提示词体系（PPT 式轮播 · 固定品牌基底）

> 目标：一条笔记的 6–9 张图，看起来像同一个号、同一个设计师做的。做法 = **锁住风格基底，只换主体画面与文字**。本项目用 **Gemini / GPT 出图**，二者能把中文文字渲染进图，所以提示词把"要显示在图上的中文"直接写进去。

## 1. 固定品牌基底（每页提示词都以它开头，逐字照搬）

一套笔记里有两类页面，**共用同一套调色板与人物卡**，只是构图密度不同：

- **情绪页**（封面 P1 / 末页 PN / 隐喻页）：柔和治愈单主体，留白勾人。
- **信息海报页**（中间的内容页）：满版知识海报，图标+文字双通道、高密度（见第 3 节开头「知识海报公共签名」）。

**品牌调色板（品牌资产，跨所有笔记复用，HEX 不随意改；要换风格只改这一处，全站统一）**：

| 用途 | 颜色 | HEX |
|------|------|-----|
| 主色调（画面/图标基调） | 雾霾蓝灰 | #A8B5C4 |
| 背景 / 暖调（纸质） | 暖米白 | #E8D8C4 |
| 辅助色 | 鼠尾草绿 | #C9D6CE |
| **强调 / 高亮**（标题关键词、结论红字、点睛） | **赭砖红** | **#A34B3A** |
| 标题主字 | 深钢蓝黑 | #2B3A4A |
| 正文 / 说明 | 深灰蓝 | #5A6B7B |

> **NBDpsy 情绪页基底**（封面/末页/隐喻页用）：柔和扁平矢量插画风（soft flat illustration），莫兰迪低饱和配色——以雾霾蓝灰 #A8B5C4、暖米白 #E8D8C4、鼠尾草绿 #C9D6CE 为主，背景暖白；圆润柔和的形体，细腻颗粒/纸质肌理，柔光、无强烈阴影，整体平静治愈；竖版 3:4 构图，主体居中或偏下，**顶部留约 1/3 空白放标题**。信息海报页的构图另见第 3 节公共签名（满版、不留大空白）。

**固定人物卡（出现人物时统一这一个形象，保证全套是"同一个人"）**：

> 圆脸、齐肩微卷短发的东亚年轻女性，穿燕麦色针织衫，神情温和平静。

> 风格基底与人物卡是**品牌资产**：跨所有笔记复用同一套，HEX 色值不要随意改。要换品牌风格时，只改这一处，全站统一。

## 2. 文字入图规范（用户用 Gemini/GPT，可渲染中文）

- **把要显示的中文用引号写进提示词**，并指定位置与层级（大标题 / 小标题 / 说明）。例：`在图片上方留白处写中文大标题"别再说自己想太多"`。
- **信息密度——每张内容页都要"有干货"**：封面页(P1)、末页(PN)保持精简（大标题 + 一句副标题即可）；**中间的内容页信息要足**——一张内容页文字总量约 80–120 字，让读者刷到时能一眼扫到"对得上自己"的那条、停下来细看，而不是只看到一句空泛标题就划走。
- **内容页版式不止"要点列表"一种——优先按内容形态选最能出干货的版式**（版式菜单见第 3 节）：
  - **要点卡（信息卡）**：并列 3–6 个平行要点时用（大标题 + 4–6 条「加粗小标题≤10字 + 一行说明≤28字」）。
  - **信息图（图文一体，密度更高、更有干货感）**：当内容有"结构关系"时，别硬塞成要点列表，改用信息图把关系画出来——**对比图**（X vs Y）、**流程/步骤图**（自救 N 步、成因链）、**象限/坐标图**（两维度分类，如依恋四象限）、**隐喻图**（抽象概念可视化，如"冰山下的情绪"）、**数据图**（占比/统计，如"约 6.8%"用圆环或人形阵列）、**拆解/结构图**（一个概念的几根支柱）。信息图靠"图形承载结构 + 短标注承载信息"，同样的干货比纯文字列表更好懂、更耐看。
- **整页信息多、但每个文字块仍短**：无论要点卡还是信息图，每个文字块（标题/标注/说明）都短、各自独立——这样 Gemini/GPT 渲染中文更准，读者也更好扫。**信息图的文字风险更高**（标注点位多）：**"≤6 字"仅指点位式短标注（如隐喻图/象限轴标）**；对比句、类型名、数据说明等各版式已在第 3 节模板里各自给了字数上限，以模板内数字为准。关键专业名词单独放大；出图后**逐字核对**；某工具长文老渲染出错，就精简标注或拆成两页。
- **N 步 / N 条清单类内容**（如"自救几步"）的取舍：**只需让读者看清步骤顺序/概览 → 用第 3 节③流程图，一页画完**（干货感强、省页数）；**每一步都要展开细节（做什么/为什么/小提示）→ 一步一卡**，每卡 ≥3 个子点把单步讲透。无论哪种，**避免只有 1–2 点的稀页**夹在足量页之间，整本密度才一致。
- **指定字体气质**而非具体商用字体名：如"圆润无衬线体""温柔手写体"，颜色用品牌深色（深灰蓝 #5A6B7B）。
- **易错专业名词的退路**：若某页必须出现 `CPTSD` `ICD-11` 等模型易渲染错的拼写/缩写，要么放大字号单独成行让模型更易写对，要么在该页提示词标注"该缩写如渲染错误请后期排版叠加"，并在"页面文字"里照常给出正确文本供人工兜底。
- 出图后**务必人工核对图上每个字**——错字是 AI 出图最常见翻车点。

## 3. 单页提示词模板

**封面页 / 末页（精简版）**：

```
【NBDpsy 风格基底 原样照搬】
画面：【本页主体场景，含固定人物卡（若有人物）+ 一两个治愈系小道具，如马克杯/绿植/毛毯/窗光】。
文字：在上方留白处写中文大标题"〇〇〇〇"（圆润无衬线体，深灰蓝色）；下方写一句副标题"〇〇〇〇"。
要求：画面干净、留白充足，竖版 3:4，不出现品牌基底之外的颜色。
```

### 知识海报公共签名（所有内容版式先叠这层，再套下面具体版式）

内容信息图页要对齐"高密度科普长图"的观感（像同一个设计师做的一套海报），每张内容页都带这 **7 个签名**，密度和"好懂"就同时上来了。**下面每个版式模板开头的「【NBDpsy 风格基底 原样照搬】」，一律理解为「情绪页基底调色板 + 本节这 7 条签名」**：

1. **纸质暖米背景 + 一圈暖棕描边框**——像加了画框的科普海报（背景 #E8D8C4 纸质肌理，四周一道暖棕细边框）。
2. **顶部一行粗黑大标题**（深钢蓝黑 #2B3A4A、思源黑体 Heavy 气质、字号最大），**标题里的关键词用赭砖红 #A34B3A 高亮**，一眼看清本页在讲什么。
3. **正文分 2–3 个区块**，每区一个小标题——用【方括号】或一个小色块 pill 底衬领起（分区标题也可用赭红点关键词）。
4. **每条信息 / 每格 / 每个节点都配一个手绘扁平小图标或简笔小人**（雾霾蓝灰 #A8B5C4 + 赭红 #A34B3A 双色、线条清晰、各条图标各不相同且贴合该条含义）——**不是装饰性小圆点**。图标承载"是什么场景"、文字承载"说什么"，**图文双通道才是密度的来源**。抽象心理概念尽量用简笔小人演绎具体场景（被忽视、逃跑、争吵、心跳加速、找咨询师…）。
5. **关键结论用赭红"红字点睛"**（如 =被抛弃 / 危险信号 / 你很安全 / 理智下线），在灰蓝正文里跳出来。
6. **因果、先后用箭头 → 连接**（经历→触发、过去→现在、第一步→第二步）。
7. **满版铺陈**——覆盖情绪页"顶部留 1/3 空白"那条：内容海报页信息铺满整版，靠**分区和区块间距**留白透气，不留大片空白。

> 一页 6–10 个信息点、每点都有自己的图标/小人，才是参考图那种"干货密度"。只有几行纯文字、没有图标演绎、没有分区和红字点睛 = 密度不足（返工）。

**内容页版式菜单——按内容形态选一种（能画成信息图就别只列要点）**：

**① 分区图标清单（信息卡）**：内容是并列 / 可分组的要点时用（对齐参考图"典型表现 / 身体信号 / 反应模式"那种分区列表）。

```
【NBDpsy 风格基底 原样照搬】
版式：知识卡海报——纸质暖米背景带一圈暖棕描边框；顶部粗黑大标题"〇〇〇〇"（关键词赭红高亮）。正文分 2–3 个区块，每块一个【方括号】小标题"〇〇"领起；每块内竖排 2–4 条，每条左侧一个雾霾蓝灰+赭红双色手绘小图标或简笔小人（各条图标各不相同、贴合该条含义），右侧加粗小标题"〇〇"+ 一行说明"〇〇〇〇"，关键结论词标赭红。
要求：满版铺陈、分区间距透气，每条图标各具辨识度、非装饰圆点，竖版 3:4，仅用品牌调色板；所有中文逐字写进图、字体清晰。
```

**② 对比图 / 对比表**：内容是"X 和 Y 有什么不同"时用（如 普通回忆 vs 情绪闪回、抑郁症 vs 抑郁型人格）。

```
【NBDpsy 风格基底 原样照搬】
版式：对比表——顶部粗黑大标题"〇〇 vs 〇〇"（两个对象名各用一色高亮：左 雾霾蓝灰、右 赭红）；下方一个两列表格，左列表头 pill"〇〇"（雾霾蓝灰底）、右列表头 pill"〇〇"（赭红/暖底）；表格 3–4 行、每行讲同一维度，左右两格各是「一个双色手绘小图标 + 一句 ≤10 字短句」左右对照；表格外有细描边、行间细分隔线。
要求：左右每行一一对应、每格都有各自的图标、层级清晰，满版竖版 3:4，仅用品牌调色板；所有中文逐字写进图、字体清晰。
```

**③ 流程 / 步骤图**：内容是"分几步 / 一环扣一环"时用（如 自救 5 步、创伤形成链）。

```
【NBDpsy 风格基底 原样照搬】
版式：竖向流程图——顶部粗黑大标题"〇〇〇〇"（关键词赭红）；下方 3–5 个圆角步骤块从上到下排列，块之间用柔和向下箭头连接，每块内左侧一个赭红序号圆点"1/2/3"、右侧加粗小标题"〇〇"+ 一行说明"〇〇〇〇〇〇"，并配一个简笔小人/图标演绎该步动作。若某步技巧较多，可在该步内嵌一个带描边的小技巧子卡，列 2–3 个编号小技巧（对齐参考图"安抚身体的技巧"子卡）。
要求：箭头流向清晰、步骤块大小一致、间距均匀、每步有图演绎，满版竖版 3:4，仅用品牌调色板；所有中文逐字写进图、字体清晰。
```

**④ 象限 / 坐标图**：内容是"按两个维度分四类"时用（如 依恋类型四象限）。

```
【NBDpsy 风格基底 原样照搬】
版式：四象限图——顶部大标题"〇〇〇〇"；中间一个十字坐标把画面分成四格，横轴两端标"〇〇/〇〇"、纵轴两端标"〇〇/〇〇"（轴标 ≤4 字）；每格内一个加粗类型名"〇〇"（≤6 字）+ 一句 ≤10 字特征，可配一个柔和小图标。
要求：四格均衡、轴标清晰、类型名醒目，竖版 3:4，仅用品牌基底配色；所有中文逐字写进图、字体清晰。
```

**⑤ 隐喻图**：抽象概念可视化时用（如"情绪像冰山，露出的只是一角"）。以画面隐喻为主、文字标注为辅。

```
【NBDpsy 风格基底 原样照搬】
画面：【一个承载概念的隐喻场景，如 水面 + 水下巨大冰山 / 一棵根系深扎的树 / 层层包裹的洋葱】，柔和扁平矢量风。
文字：顶部大标题"〇〇〇〇"；在隐喻的关键部位旁做 2–3 处 ≤6 字短标注（如冰山露出部分标"你看到的"、水下标"你没看到的"），引线细柔。
要求：隐喻一眼看懂、标注不遮画面、留白透气，竖版 3:4，仅用品牌基底配色；所有中文逐字写进图、字体清晰。
```

**⑥ 数据图**：内容含占比 / 统计时用（如"约每 6 人有 1 人经历过…"）。

```
【NBDpsy 风格基底 原样照搬】
版式：数据可视化——顶部大标题"〇〇〇〇"；中间用【圆环占比图 / 一排人形图标（部分高亮）/ 简洁柱状】表现核心数字"〇〇%"或"N 分之 1"，数字放大醒目；下方一行 ≤16 字说明这个数字意味着什么 + 一行灰色小字标数据出处名。
要求：数字最醒目、图形简洁不花哨、出处清楚，竖版 3:4，仅用品牌基底配色；所有中文逐字写进图、字体清晰。
```

**⑦ 拆解 / 结构图**：内容是"一个概念由几个部分/几根支柱组成"时用（如 凝聚自体的三根支柱、安全感的构成）。

```
【NBDpsy 风格基底 原样照搬】
版式：拆解结构图——顶部大标题"〇〇〇〇"（要拆解的整体概念）；下方用一个统摄图形把 3–4 个组成部分组织起来（如 顶部一个大圆"〇〇"下面伸出 3 根柔和支柱、或一个中心圆向外散出 3–4 个小圆），每个部分内加粗小标签"〇〇"（≤6 字）+ 其下一行 ≤12 字说明；部分之间用细柔连线示意从属关系。
要求：整体与部分的层级一眼可辨、连线清晰不杂乱、留白透气，竖版 3:4，仅用品牌基底配色；所有中文逐字写进图、字体清晰。
```

**⑧ 因果链图**：内容是"某段经历 → 现在的触发 → 导致的结论"时用（如 原生家庭印记 / 过往情伤如何触发情绪闪回）。可单列，或左右双栏各讲一组来源。

```
【NBDpsy 风格基底 原样照搬】
版式：因果链海报——顶部粗黑大标题"〇〇〇〇"（关键词赭红）。下方 1–2 组因果链（双栏时左右各一组），每组一个色块底小标题"〇〇"领起；链条形式为「一个手绘场景小人插画 + "〇〇经历" → 手绘场景 + "〇〇触发" ＝ 赭红结论词"〇〇"」，箭头清晰；每组下方可加一条色块底总结句"〇〇〇〇"。底部可通栏放一句 Tips。
要求：每一环都有手绘小人演绎具体场景、箭头流向清晰、结论词赭红最醒目，满版竖版 3:4，仅用品牌调色板；所有中文逐字写进图、字体清晰。
```

> **选版式的判断**：内容有"对照/先后/分类/占比/构成/因果关系"→ 选对应信息图（②–⑧），干货感最强；只是几条并列 tips → 用分区图标清单（①）。一套 6–9 页里**混用多种版式**比全是要点卡更耐刷；但风格基底、调色板、人物卡始终统一。

**封面页（P1）额外要求**：标题字号最大最醒目、画面要有记忆点、与笔记标题呼应——它决定点击率，单独打磨；封面不堆要点，留悬念把人勾进来。

**末页（PN）额外要求**：放行动建议 + 品牌锚句一行（"NBDpsy · 北大硕博纯线上心理咨询"）+ 危机声明小字（"心理科普，非诊断治疗 · 援助热线 12356"）。**不放任何二维码/微信/链接**（导流红线）。

## 4. 一套 6–9 张如何保持风格一致

- **风格基底逐字不变**，每页只改"画面主体"和"文字"两行。
- **第 1 张定稿后作参考图**：Gemini/GPT 支持上传参考图，把定稿的 P1 喂进去让后续页"保持同一画风与人物"，比纯文字复现更稳。
- 主色 HEX 在每页都带上，复制粘贴时不要改。
- 人物在多页出现时，每页都带固定人物卡描述，避免长相漂移。

## 5. 范例（单页 · 中文 · 文字入图）

**P1 封面**（CPTSD 主题）：

```
柔和扁平矢量插画风，莫兰迪低饱和配色——雾霾蓝灰 #A8B5C4、暖米白 #E8D8C4、鼠尾草绿 #C9D6CE 为主，背景暖白；圆润柔和形体，细腻颗粒肌理，柔光无强阴影，平静治愈；竖版 3:4，主体居中偏下，顶部留约 1/3 空白。
画面：一位圆脸、齐肩微卷短发的东亚年轻女性，穿燕麦色针织衫，独自坐在窗边捧着温热的马克杯望向柔光，身旁一盆小绿植，神情温和平静。
文字：在上方留白处写中文大标题"总说自己想太多？"（圆润无衬线体，深灰蓝 #5A6B7B，字号最大）；其下写副标题"也许是复杂性创伤"（同色，字号略小）。
要求：画面干净、留白充足，竖版 3:4，仅用品牌基底配色。
```

## 6. 视频参考图提示词（去文字版 · 页对页）

每篇笔记文件末尾追加一节 `## 视频参考图提示词`，为轮播的每一页给出一条"去文字版"提示词。

- **用途**：nbdpsy-text-to-video 图生模式用这套提示词生成"无文字同款图"，当每个镜头的首帧参考图。视频阶段会把旁白烧录成字幕——图内若再有文字，会与字幕重叠打架，所以参考图必须完全无文字。本节只产提示词，图由 nbdpsy-text-to-video 产线按需生成（该产线与出图环节同一套宿主自适应逻辑）。
- **推导规则**：拿该页的绘图提示词，**剔除所有图内文字指令**（「文字：…」「顶部大标题"…"」以及基底里"顶部留出约 1/3 空白用于放标题文字"这类为文字服务的构图指令，可改为"构图完整、留白自然"）；**保留品牌基底（三主色 HEX）、固定人物卡、画面主体与版式氛围**；末尾追加一句**「画面中无任何文字」**。
- **页对页**：P1 对 P1、PN 对 PN，画面主体与轮播版**语义呼应**（要点卡页去文字后结构相仿；信息图页有合理例外，见下）。不同版式的去文字化各有讲究：
  - **要点卡页**：把"文字条目"改述为等量的图形元素（小圆图标竖排等），保持画面结构相仿。
  - **信息图页**（对比/流程/象限/数据/拆解）：**别保留没文字的空图表骨架**（空坐标轴、空箭头没有意义、也不适合当视频首帧）；改为回到这一页的**内核场景或隐喻**——用固定人物卡的一个相关情绪画面、或该概念的隐喻意象（如对比页→一个人站在两条岔路前；流程页→拾级而上的台阶；数据页→人群中一个被柔光标出的身影），保留品牌基底与氛围。
  - **隐喻图页**：本就以画面为主，去掉标注即可，隐喻场景原样保留。
- **格式硬约束**：节内每页用加粗标记 `**P1**`…`**PN**` + 围栏提示词；**绝不用 `### PN` 标题**——`### PN` 是「## 配图轮播」的页数计数契约（count_xhs.py 全文数 `### P*`），在本节复用会让页数翻倍误判。
