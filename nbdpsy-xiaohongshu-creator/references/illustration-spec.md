# 插图提示词体系（PPT 式轮播 · 固定品牌基底）

> 目标：一条笔记的 6–9 张图，看起来像同一个号、同一个设计师做的。做法 = **锁住风格基底，只换主体画面与文字**。本项目用 **Gemini / GPT 出图**，二者能把中文文字渲染进图，所以提示词把"要显示在图上的中文"直接写进去。

## 1. 固定品牌基底（每页提示词都以它开头，逐字照搬）

> **NBDpsy 风格基底**：柔和扁平矢量插画风（soft flat illustration），莫兰迪低饱和配色——以雾霾蓝灰（#A8B5C4）、暖米白（#E8D8C4）、鼠尾草绿（#C9D6CE）为主，背景暖白；圆润柔和的形体，细腻颗粒/纸质肌理，柔光、无强烈阴影，整体平静治愈；竖版 3:4 构图，主要画面居中或偏下，**顶部留出约 1/3 空白用于放标题文字**。

**固定人物卡（出现人物时统一这一个形象，保证全套是"同一个人"）**：

> 圆脸、齐肩微卷短发的东亚年轻女性，穿燕麦色针织衫，神情温和平静。

> 风格基底与人物卡是**品牌资产**：跨所有笔记复用同一套，HEX 色值不要随意改。要换品牌风格时，只改这一处，全站统一。

## 2. 文字入图规范（用户用 Gemini/GPT，可渲染中文）

- **把要显示的中文用引号写进提示词**，并指定位置与层级（大标题 / 小标题 / 说明）。例：`在图片上方留白处写中文大标题"别再说自己想太多"`。
- **信息密度——每张内容页是一张"知识卡"**：封面页(P1)、末页(PN)保持精简（大标题 + 一句副标题即可）；**中间的内容页要有料**——大标题(≤14 字) + **4–6 条要点，每条 = 加粗小标题(≤10 字) + 一行说明(≤28 字)**。这样读者刷到时能一眼扫到"对得上自己"的那条、停下来细看，而不是只看到一句空泛的标题就划走。一张内容页文字总量约 80–120 字——信息密度要够，别让一页只有一句空泛标题。
- **整页信息多、但每个元素仍短**：虽然一页装 4–6 条，但每个文字块（大标题/小标题/说明）都短、各自独立成行——这样 Gemini/GPT 渲染中文更准，读者也更好扫。出图后**逐字核对**；若某工具长文老渲染出错，再把该页说明精简或拆成两页。
- **N 步 / N 条清单类内容**（如"自救 5 步"）：优先**一步一卡**——每卡配 3 个子点（如 做什么 / 为什么 / 小提示），把单步讲透；若要把多步并到一卡，也要凑足 ≥3 个信息点。**避免出现只有 2 点的稀页**夹在足量信息卡之间，整本密度才一致。
- **指定字体气质**而非具体商用字体名：如"圆润无衬线体""温柔手写体"，颜色用品牌深色（深灰蓝 #5A6B7B）。
- **易错专业名词的退路**：若某页必须出现 `CPTSD` `ICD-11` 等模型易渲染错的拼写/缩写，要么放大字号单独成行让模型更易写对，要么在该页提示词标注"该缩写如渲染错误请后期排版叠加"，并在"页面文字"里照常给出正确文本供人工兜底。
- 出图后**务必人工核对图上每个字**——错字是 AI 出图最常见翻车点。

## 3. 单页提示词模板

**封面页 / 末页（精简版）**：

```
【NBDpsy 风格基底 原样照搬】
画面：【本页主体场景，含固定人物卡（若有人物）+ 一两个治愈系小道具，如马克杯/绿植/毛毯/窗光】。
文字：在上方留白处写中文大标题"〇〇〇〇"（圆润无衬线体，深灰蓝色）；下方写一句副标题"〇〇〇〇"。
要求：画面干净、留白充足，竖版 3:4，不出现品牌基底之外的颜色。
```

**内容页（信息卡 · 信息量大的主力版式）**：

```
【NBDpsy 风格基底 原样照搬】
版式：信息卡（知识卡）布局——顶部居中大标题"〇〇〇〇"；下方竖向排列 4–6 条，每条左侧一个柔和小圆图标，右侧加粗小标题"〇〇"、其下一行浅灰说明"〇〇〇〇〇〇"；条目左对齐、间距均匀、层级清晰。可在卡片底角放一个小小的固定人物卡剪影或治愈小道具点缀（不抢文字）。
要求：信息清楚但不拥挤、留白透气，竖版 3:4，仅用品牌基底配色；所有中文逐字写进图、字体清晰。
```

**封面页（P1）额外要求**：标题字号最大最醒目、画面要有记忆点、与笔记标题呼应——它决定点击率，单独打磨；封面不堆要点，留悬念把人勾进来。

**末页（PN）额外要求**：放行动建议 + 品牌锚句一行（"NBDpsy · 北大硕博纯线上心理咨询"）+ 危机声明小字（"心理科普，非诊断治疗 · 援助热线 12356"）。**不放任何二维码/微信/链接**（导流红线）。

## 4. 一套 6–9 张如何保持风格一致

- **风格基底逐字不变**，每页只改"画面主体"和"文字"两行。
- **第 1 张定稿后作参考图**：Gemini/GPT 支持上传参考图，把定稿的 P1 喂进去让后续页"保持同一画风与人物"，比纯文字复现更稳。
- 主色 HEX 在每页都带上，复制粘贴时不要改。
- 人物在多页出现时，每页都带固定人物卡描述，避免长相漂移。

## 5. 范例（单页 · 中文 · 文字入图）

**P1 封面**（CPTSD 主题）：

```
柔和扁平矢量插画风，莫兰迪低饱和配色——雾霾蓝灰 #A8B5C4、暖米白 #E8D8C4、鼠尾草绿 #C9D6CE 为主，背景暖白；圆润柔和形体，细腻颗粒肌理，柔光无强阴影，平静治愈；竖版 3:4，主体居中偏下，顶部留约 1/3 空白。
画面：一位圆脸、齐肩微卷短发的东亚年轻女性，穿燕麦色针织衫，独自坐在窗边捧着温热的马克杯望向柔光，身旁一盆小绿植，神情温和平静。
文字：在上方留白处写中文大标题"总说自己想太多？"（圆润无衬线体，深灰蓝 #5A6B7B，字号最大）；其下写副标题"也许是复杂性创伤"（同色，字号略小）。
要求：画面干净、留白充足，竖版 3:4，仅用品牌基底配色。
```

> Gemini/GPT 出图后，导出 1080×1440（3:4），人工核对图上"总说自己想太多？""也许是复杂性创伤"无错字，即可作封面。

## 6. 视频参考图提示词（去文字版 · 页对页）

每篇笔记文件末尾追加一节 `## 视频参考图提示词`，为轮播的每一页给出一条"去文字版"提示词。

- **用途**：nbdpsy-text-to-video 图生模式用这套提示词生成"无文字同款图"，当每个镜头的首帧参考图。视频阶段会把旁白烧录成字幕——图内若再有文字，会与字幕重叠打架，所以参考图必须完全无文字。本节只产提示词，图由 nbdpsy-text-to-video 产线按需生成（该产线与出图环节同一套宿主自适应逻辑）。
- **推导规则**：拿该页的绘图提示词，**剔除所有图内文字指令**（「文字：…」「顶部大标题"…"」以及基底里"顶部留出约 1/3 空白用于放标题文字"这类为文字服务的构图指令，可改为"构图完整、留白自然"）；**保留品牌基底（三主色 HEX）、固定人物卡、画面主体与版式氛围**；末尾追加一句**「画面中无任何文字」**。
- **页对页**：P1 对 P1、PN 对 PN，画面主体与轮播版一致，只是没有文字。信息卡版式页可把"文字条目"改述为等量的图形元素（小圆图标竖排等），保持画面结构相仿。
- **格式硬约束**：节内每页用加粗标记 `**P1**`…`**PN**` + 围栏提示词；**绝不用 `### PN` 标题**——`### PN` 是「## 配图轮播」的页数计数契约（count_xhs.py 全文数 `### P*`），在本节复用会让页数翻倍误判。

**模板**：

````markdown
## 视频参考图提示词

> 去文字版，页对页；供 nbdpsy-text-to-video 图生模式当首帧参考图。视频会烧录字幕，图内不能有任何文字。

**P1**
```
【NBDpsy 风格基底 原样照搬（去掉"顶部留白放标题"一句）】
画面：【与轮播 P1 相同的主体场景与人物】。
要求：画面中无任何文字（无标题、无水印）；构图完整、留白自然，竖版 3:4，仅用品牌基底配色。
```

**P2**
```
…
```
````
