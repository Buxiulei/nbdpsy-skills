# 小红书笔记规格 + 合规红线（心理科普类）

> 依据：小红书平台规则、《广告法》§9（极限词）/§17（医疗用语）、《交易导流违规管理细则》（2025-03-12 生效）、全国统一心理援助热线 12356（2025-05-01 前并线）。下列"硬性要求"违反任一即不合格。

## 1. 笔记结构规格

### 标题（决定点击的第一道关）
- **硬限 20 字**，重点前置。结构常见为「关键词 + 钩子词」。
- 四类钩子任选其一：
  - **数字体**：「3 个迹象说明你可能…」「5 步自救」
  - **痛点体**：「总说自己想太多？」「为什么我总是 emo」
  - **悬念体**：「我后悔没早点知道这件事」
  - **身份标签**：「i 人必看」「留学生码住」
- 符号 `｜``【】` + 1–3 个 emoji 点缀，不堆砌。

### 正文 —— 三段式骨架（**本项目强制结构**）

> **为什么必须是三段式**：纯科普易涨粉、难转化（读者收藏完就走，不知道你能帮他）；业务硬广易转化、难曝光（平台限流 + 投流不过审 + 读者反感）。**「痛点场景开头 + 科普干货主体 + 结尾轻引导」是两难之间唯一的解**——它同时对齐小红书用户的真实决策链路（痛点搜索 → 案例对比 → 私信咨询 → 社群成交），也符合投流审核的通过规律（全文绝大部分是科普干货、只有结尾一句轻引导的笔记更容易过审；通篇引导或开头即广告的，限流与拒审风险最高）。

| 段 | 占比 | 写什么 | 硬要求 |
|---|---|---|---|
| **① 痛点场景开头** | 约 15–20% | 用一个**具体的生活场景**（不是抽象定义）让读者"对号入座"：某个时刻、某句话、某种身体反应。首句就是 3 秒钩子。 | 场景必须具体可感（"凌晨三点又醒了，脑子里循环白天那句话"），**禁止**用"你是否常常感到焦虑？"这种问卷腔 |
| **② 科普干货主体** | 约 65–75% | 这是笔记的**主体和价值来源**：解释机制、给判据、给可执行的方法。分点写，每点都有信息量。 | 干货必须能独立成立（**读者不点你主页也有收获**）；数字/结论可溯源到源长文；这一段**完全不谈自家服务** |
| **③ 结尾轻引导** | 约 10%，**1–2 句** | 先给互动 CTA，再**轻**提一句"还有专业支持这条路"。**轻 = 陈述事实、给出选项，不催不促不承诺**。 | 见下方「轻引导写法」；**禁止**任何硬广特征（限时/优惠/名额/立即/私信我领），**禁止**站外导流（微信/二维码/外链） |

**轻引导写法（结尾这 1–2 句怎么写）**：

- ✅ 安全句式（陈述 + 留门）：
  - 「如果这些方法试过还是走不出来，那不是你不够努力——有时候只是需要一个专业的人陪你一起看。」
  - 「自我调节能解决一部分，剩下的部分交给专业陪伴也不丢人。主页有更多这个主题的内容。」
  - 「觉得有用就收藏。想聊聊自己的情况，评论区或站内私信都可以。」
- ❌ 反面（硬广特征 / 违规）：
  - 「限时优惠，现在预约立减 200」（硬广 + 促销）
  - 「私信我领取免费测评名额」（诱导 + 名额）
  - 「加微信 xxx 免费咨询」（站外导流，直接封号）
  - 「我们能治好你的焦虑」（疗效承诺 + 医疗违禁）
- **品牌锚句每篇只出现一次**（见 §3），放在结尾轻引导里自然带过，不重复刷。
- ⚠️ **最低出口律（§1.5 G1，全路径适用）**：轻引导之前（主体末或结尾），必须有**一个今天就能做的小动作**，且**门槛低于付费**——只给共鸣不给出口的笔记（尤其纯金句型）是"高唤起 + 零效能"，读者会关掉手机而不是求助。
- ⚠️ **CTA 黑名单（§1.5 G7）**：「私信我」「扣 1」「加我」「主页链接」「限时」「名额」「仅剩」及其谐音变体、任何"领取"式利益诱饵，一律禁用。上面 ✅ 的「评论区或站内私信都可以」之所以安全，是因为它**陈述选项、不给诱饵**。
- ⚠️ **危机声明与 CTA 分离（§1.5 G6）**：12356 那一行**单独成段**，不与 CTA 同句同段；笔记内容本身触及危机识别与转介时，**当篇不放商业 CTA**。

**其它硬性要求**：
- **硬限 1000 字**；本项目**默认 300 汉字**（用户指定，只数正文主体）。小红书爆款经验区间为 500–800 字（更厚更易被收藏）——如要提收藏率可调到 500–600，但默认遵守 300。三段式在 300 字下的典型配比：开头 50–60 字 / 主体 200–220 字 / 结尾 30–40 字。
- 主体分段与 emoji 密度按 **§1.6 口径**执行（每段 1–3 短句不超 3 行；emoji 全文 3–6 个且温和款）；分点用 `·` 或编号。
- 口语化、第一人称、像朋友说话；**共情先于科普**（这是心理内容区别于普通种草文的关键）。
- **结尾 CTA**（在轻引导之前或之内）：提问引评论 / 「收藏不迷路」/ 「评论区抱抱」。

### 话题标签 #
- **硬限 10 个**，建议 **5–10 个**，三层各 2–4 个：
  - **关联**（精准品类/品牌词）：`#心理科普` `#CPTSD` `#情绪内耗`
  - **热度**（蹭当下热点话题）
  - **长尾**（细分 + 身份/地域）：`#留学生心理` `#童年创伤` `#上海心理咨询`
- **位置**：作为可见文本 `#标签1 #标签2 …` 附在「发布文案」正文末尾（小红书惯例标签贴文末），**同时**写进 post 文件 frontmatter 的 `hashtags` 备查。只放 frontmatter 渲染时看不见、也没法复制，等于没标签。

### 图文轮播
- **硬限 18 张**（2024 年由 9 张扩容到 18），**建议 6–9 张**（经验值，无官方最优数）。
- **比例 3:4 竖版，1080×1440px**——显示面积比横版约大 40%、点击率最高。全套统一尺寸，防止滑动时跳动。
- **首图 = 门面**：大标题 10–15 字、字号醒目；安全区距顶 ≥150px、距底 ≥200px、左右各 ≥60px（关键文字别贴边，避免被 UI 遮挡）。
- 每张当 1 页 PPT：1 个要点 = 1 个大标题 + 4–6 条短点（信息密度要够，别只放一句空泛标题）。

### 选题口吻（心理/情绪类爆款角度）
内核：**情绪背后是未被满足的需求**。三种最稳的体裁：
1. **第一人称共情体**——"如果你也常常…，这篇写给你"。
2. **干货清单体**——编号清单，天然适合收藏。
3. **自救指南体**——痛点 + 可执行的小步骤。
热门母题：人际关系、心理学效应、恋爱心理、缓解内耗/焦虑、自我觉察。

## 1.5 选题的生死线：从「现象」→「场景」→ 唯一的一条「深挖路径」（本项目最高优先级规则）

> **为什么这条最重要**：停留在现象层的笔记，读者的反应是"学到了"（收藏 → 划走 → 忘记）；深挖到场景层的笔记，读者的反应是**"这说的就是我"**（对号入座 → 情绪被击中 → 才可能付费）。
> **付费不是被科普说服的，是被场景击中的。** 科普只提供"可信"，场景提供"关我什么事"。
> **但"被击中"只是必要条件，不是充分条件。** 一个已经痛到极点、却被"我不好意思 / 我试过没用 / 我不知道找谁 / 我一直没约"钉住的读者，你再给他一篇更痛的场景文，他只会**更痛地划走**。
> **所以每篇笔记要回答两个问题：① 它击中哪个场景？② 它把读者往前推了哪一格？** 第 ② 个问题的答案，就是这篇的**深挖路径**。

### 现象 vs 场景（一眼看懂差别）

| | ❌ 现象层（泛，无痛感） | ✅ 场景层（具体，有痛感） |
|---|---|---|
| 例 | 孩子内向不爱说话 | 孩子在学校被欺负了，回家也一个字不说——**你问什么他都摇头，你试了三种方法，他把门关得更紧** |
| 例 | 亲密关系中的沟通问题 | 你们已经三个月没吵架了——**但也三个月没聊过一句超过五分钟的话**，这种"和平"比吵架更让你心慌 |
| 例 | 职场焦虑 | 周日下午四点，你开始胃疼、反复看手机——**明天要开周会，而你已经这样过了十七个周日** |
| 例 | 原生家庭创伤 | 你妈只是打来一个再普通不过的电话，你却在挂断后蹲在阳台上哭了半小时——**你自己都说不清为什么** |
| 例 | 线上心理咨询很方便 | 你在纽约做了两年英文咨询，进步很大——**可你始终没法用英文讲清楚，我妈那句"我都是为你好"到底是什么味道** |

**判据（一票否决）**：如果这个选题**换成任何别的行业/人群也说得通**，它就还停在现象层。"孩子内向不爱说话"教育机构、口才班、儿童摄影都能写；而"被欺负了回家一个字不说、你试了三种方法他把门关得更紧"只有心理这条路能接住。

### 分类基准（唯一划分维度）：**这篇笔记消解的那一道求助闸门**

从"痛"到"约"之间横着一排闸门——不承认在痛 / 说不清 / 归错因 / 觉得不值得处理 / 觉得该自己扛 / 试过没用 / 凭什么花这个钱 / 不知道找谁 / 怕第一次 / 约不到 / 一直没约。**一篇笔记只干一件事：拆掉其中一道。** 拆掉的那一道 = 它的**主路径**。

**为什么用这个维度**（其它三个候选都评估过了，别再重开）：

| 候选基准 | 判决 | 理由 |
|---|---|---|
| **求助闸门**（本表采用） | ✅ | 唯一直接回答"读者为什么还没来找我们"的维度；移动一格 = 向付费前进一步；危机内容在链上无处安放（我们无胜任力跨过它）→ 结构性挡在选题池外 |
| 时间位置（过去/当下/未来） | ❌ 降为**篇内叙事模板** | 一篇合格的笔记天然横跨三段（小时候→现在→再这样下去），三段是同一条叙事的部件，不是三种笔记 |
| 情绪杠杆（恐惧/羞耻/愤怒…） | ❌ 降为**合规监控标签** | 情绪共现（"孩子被欺负不说话"同时是恐惧+羞耻+愤怒+无力），强行选一个 = 阉割场景；且把"恐惧"立为一级类目 = 在制度上给恐惧型选题发配额 |
| 叙事视角（自述/旁观/对话/数据） | ❌ 降为 **A/B 变量** | 这是表达形式不是场景类型；换视角不改变读者是否被击中 |

**⚠️ 分类的对象是「笔记」，不是「读者」。** 不要问"读者卡在哪"——你看不见读者，同一篇笔记会被十种状态的人读到，这个问题没有唯一答案。要问**"这篇笔记的落点在哪"**：落点写在文本里，是可核验的物证，两个人能对着同一段文字复核。

**落点判定（唯一规则，30 秒判完）**：

> **看最后三行（不含危机声明与标签行）：读完这篇，作者希望读者完成的那一个内在动作是什么？**
>
> 认领痛感 / 给处境命名 / 换掉一个归因 / 看见代价 / 卸下自责 / 明白为什么试过没用 / 想清楚凭什么花这个钱 / 知道怎么选 / 预演第一次 / 破解约不到 / **今晚就把它约掉**。
>
> **一个动作 = 一条主路径。数不出唯一的那一个 = 这篇没想清楚，返工。**

不允许写"主路径：P2 + P4"。可以加**次要路径 ≤1 条**，但 **P3 / P4 不得作次要路径**（见 G8——否则恐惧内容会以"次要路径"的名义绕过配额与护栏）。

**标注格式**（写进每篇 post 的 frontmatter，同时在 `00-overview.md` 每篇一行）：

```yaml
scene_path: P2 处境命名              # 主路径，唯一，必填
scene_path_secondary: P5 去羞耻化    # 可空；最多 1 条；不得填 P3 / P4
audience: 当事人                     # 当事人 / 身边人（关系人轴，见下）
chronicity: 慢性关系型               # 慢性 / 单点急性（单点急性必须写明理由，默认打回）
increment: 经验                      # 信息 / 经验 / 整理 —— 三者全无 = 毙
```

### 路径总表（12 条 · 三层 · **层内无序**）

链是**分段偏序**：层与层之间大体有先后，**层内并列、不比大小**。读者是可以跳闸的——医生一句"建议配合心理咨询"就能让人从认知层直接跳进转化层（这是站内最硬的付费扳机之一）。**所以没有"从 P1 开始逐个排除"的遍历规则，只有落点判定。**

| # | 路径 | 读者卡点（第一人称） | 客户质量 | 流量场 | 一套里的配额 | 合规风险 |
|---|---|---|---|---|---|---|
| **P1** | 痛感认领 | "大家不都这样吗，我只是有点累" | 不选客·**几乎不转化** | 推荐流（涨粉王） | 弹性 ≤1 | 中 |
| **P2** | **处境命名** ★主力 | "我说不上来哪里不对" | **★★★ 长期客**（唯一被验证的闭环） | 推荐流 | **固定 2**（8 篇套可 3） | 低 |
| **P3** | 归因矫正 | "我就是懒 / 是他有问题" | ⚠️ **取决于落点** | 推荐流 | 弹性 ≤1 | **中高** |
| **P4** | 代价显影 | "忍忍就过去了，还没到那个地步" | ⚠️ **钩买止痛的脱落客** | — | **生产配额 = 0** | **高** |
| **P5** | 去羞耻化 | "是我不够好 / 该我自己扛 / 说出来像有病" | ★★ 长期客 | 推荐流 | **固定 1** | 中 |
| **P6** | 失效归因 | "我试过了，没用" | ★（证据待验证） | 推荐 + 搜索 | 弹性 ≤1 | 中 |
| **P7** | **价值论证 / 替代方案** | "跟 AI 聊两小时、跟闺蜜吐槽、自己看书好像也行——凭什么花几百块 × 十几次？" | 待验证（**2026 头号缺口**） | 搜索为主 | 弹性 ≤1 | 中 |
| **P8** | 求助路径显影 | "然后呢？找谁？怎么挑？值不值？什么时候自助就够？" | ★★☆ 高客单 | 搜索为主 | **固定 1**（取证前可退回 P2） | 中 |

| **F1** | ⛔ 治疗方案科普 | （疗法对比 EMDR/图式/IFS、服药与咨询并行） | — | — | **公域冻结** | **法律级** |
| **PX** | 带他进门（**关系人轴专属**） | "我知道他需要，但他不肯去" | 待验证 | 推荐 + 搜索 | 弹性 ≤1 | 中 |

```

阻力层  P4 代价显影(禁产) · P5 去羞耻化 · P6 失效归因 · P7 价值论证      ← 层内无序
转化层  P8 求助路径显影 · P9 首访祛魅 · P10 可得性破解 · P11 临门推动
冻结区  F1 治疗方案科普（只落官网 + 咨询师实名署名 + 权威源，待法务拍板）
正交轴  受众身份 = 当事人 / 身边人；身边人轴多一道 PX，且九闸门语义重写
```

**受众轴（定义域参数，不是第 13 条路径）**：读者 ≠ 受苦者时（父母替孩子、伴侣替对方、子女替父母），闸门要**重新映射到读者自己身上**：她的痛是"我看着他难受，我无能为力"（这本身就是一个 P2 处境），她的动作是"我怎么开口 / 我要不要自己先来一次"（PX）。
> **硬规则**：身边人内容的落点**必须是读者自己能做的一个动作**（先照顾好你自己 / 先说你自己 / 你可以先来一次），**绝不能是"如何让对方改变"**。落在"三招让孩子开口 / 让女朋友冷静" = **他者解码 = 毙**（它把注意力从"我们的模式"引向"怎么操作对方"，等于关闭求助链）。
> **危机边界**：身边人轴上，若**受苦者**处于危机 / 重性精神障碍（自伤自杀、双相、精神病性）→ 同样排除，只给 12356。**但"照顾者本人"（照顾者倦怠、内疚、边界丧失）是我们欢迎的来访者**——排除的判据是"当事人处于我们无胜任力的状态"，**不是"提问的人是家属"**。

> 📎 编号对照调研稿 v1（九路径）：v1 P7 → 本表 **P8 + F1**（拆分）；v1 P8 → **P9**；v1 P9 → **P10**；新增 **P7 价值论证 / P11 临门推动 / PX 带他进门**。

### 逐条路径详解（公式 · 例 · 写砸 · 护栏）

#### P1 · 痛感认领 ｜ 弹性 ≤1 ｜ 涨粉，不转化
- **公式**：一句具体的白描处境 ＋ 一句"你不用解释"的认领 ＋ **人群锚点 + 慢性度锚点**（让共鸣的人落在长期客画像里）＋ **一个今天能做的小动作**（G1 最低出口律）
- **✅ 例**：「你不是懒。你只是每天下班后要在车里坐二十分钟，才有力气上楼——**这样的晚上已经太多了的话，今天可以只做一件小事：跟一个人说一句'我最近不太好'，哪怕对方只回一个'嗯'。**」
- **❌ 写砸**：① **给共鸣不给出口**（金句写完直接结束）——对孤独/绝望读者是"高唤起 + 零效能"，我们制造了一次共鸣，没有制造一次求助；② 空泛疗愈鸡汤（"成为爱的源头，整个世界都会映照你的光芒"）→ 无增量 = 毙；③ 滥用——每篇都写成金句，账号会被算法认成情绪号
- **⚠️ 负外部性（老板已知的取舍）**：P1 钩来的是**情绪消费者**（点赞收藏但不预约）。账号的算法画像是**跨篇累积**的，靠"每套 ≤1 篇"关不住——一旦互动画像被情绪人群占据，后续的转化层笔记会被优先推给他们。**监控指标：账号级收藏/点赞比**（>1.5 = 实用可转化；<1.0 = 情绪号）。**连续下滑 → 立刻停 P1。**

#### P2 · 处境命名 ｜ 固定 2（8 篇套可 3）｜ ★★★ 长期客·主力
- **公式**：具体处境（谁 ＋ 什么时刻 ＋ 那一句话 / 那个身体反应）＋ **计数器**（"第十七个周日""每个月都要经历一次"）＋ 给它一个**可工作的议题名**（不是诊断名）＋ 一个今天能做的小动作
- **✅ 例**：「你妈只是打来一个再普通不过的电话，你却在挂断后蹲在阳台上哭了半小时——你自己都说不清为什么。这一幕，你每个月都要经历一次。」
- **❌ 写砸**：① 退回现象层（"亲密关系中的沟通问题"）= 一票否决；② **滑成标签测评**（"你是讨好型人格吗""对上 3 条就是 CPTSD"）——那是 P3 的病理化形态，产出的是**"自诊完成态"不是购买态**（生产库硬数据：预沟通不转化的 42 人里创伤/CPTSD 词占 49%，其中 8 人明确写出"CPTSD"字样、**零转化**；站内 CPTSD 集群 11 篇日均浏览 3.86 全站最低、零预约关联）；③ 写成**单点急性**（"昨天他跟我提了分手"）→ 钩来 48 小时内下单、痛感一降就走的止痛客
- **护栏**：命名的是**议题**（"你和母亲之间的边界一直没建立起来"），不是**诊断**（"你有 CPTSD"）；**术语（CPTSD / 回避型依恋 / 讨好型人格）最多在正文出现一次作解释性名词，不得进标题、不得做落点**；涉躯体症状必须挂分流句（G2）
- **硬证据**：原生家庭议题占比——长期客 70% vs 脱落客 23% vs 预沟通不转化 13%（跨组干净）；站内**唯一被验证的"内容 → 付费"闭环**（《依恋修复指南》浏览第 1、读完率 71% 第 1 → 预约咨询师本人）

#### P3 · 归因矫正 ｜ 弹性 ≤1 ｜ ⚠️ 降权使用（**落点决定生死**）
- **公式**：你以为是 X（那个让你不必求助的解释）＋ 其实更可能是 Y（**一个机制，不是一个病名**）＋ 2–3 条可自我验证的线索 ＋ **"知道了这个，你今天可以先做什么"**
- **✅ 例**：「你在查'回避型依恋怎么办'。但真正卡住的可能不是他回避——是他一沉默，你就必须立刻确认'我们还好吗'，而这份确认越急，他退得越远。（也可能只是你们最近都太累了。但如果这一幕重复了很多次，它就不只是累。）**下次他沉默时，试着先记录你身体的第一反应，而不是立刻发消息。**」
- **❌ 写砸（两种，都是绝杀）**：① **贴诊断标签**（"你以为你只是内向，其实你是 CPTSD"）→ 自诊完成态，零转化（数据同上）；② **给不在场的第三人贴标签**（"你妈是隐性 NPD 的 7 个信号""他是回避型""原生家庭 PUA"）→ 无诊断资质 + 制造代际/性别对立（平台 2025 年因此处置笔记 459.8 万篇）+ 不可证伪。**流量、口碑、转化三输。**
- **护栏（四条，缺一即打回）**：**① 非诊断律**（给机制不给病名）；**② 基率诚实律**（主动给出替代解释"也可能只是最近太累了"；用"更可能/有研究提示"不用"你就是"）；**③ 不制造对立律**（写处境不写反派）；**④ 不给第三人贴标签律**（读者的父母/伴侣/子女/上司**不是我们的诊断对象**；P3 的落点永远是"**你们之间的那个模式**"，不是"**他是什么人**"）
- **落点闸门**：终点是"确认标签" → **返工**；终点是"机制 + 下一步" → 放行。**P3 若入选，必须与 P8 / P9 配对投放**（否则读者停在"我知道我是什么了"就走了）

#### P4 · 代价显影 ｜ ⛔ **生产配额 = 0（默认不做）** ｜ 分类保留，作**废稿检测器**
- **为什么禁产**：① 客户质量最差——投放包「急性危机/失恋挽回」加权 4.05 分（8 个包垫底）已被内部数据完整证伪（均值 1.5 次、净 ARPU 578 元、只占 2.2% 收入）；② 系统性钩入**平台接不了的超范围个案**（占预沟通不转化的 11.9%：双相、幻听、自杀意念现症——**不但不付钱，还倒扣咨询师工时**）；③ **五条护栏执行后它唯一的优势（抓人）就没了**，于是在流量 KPI 下运营必然向红线漂移
- **落进 P4 = 打回**：把它改写成 P2 的慢性版（加计数器），或换题。**确有必要做 → 单篇书面审批 + 咨询师签字，不进常规配比。**
- **一旦获批，五条护栏全过才能发**：① 只写**功能性/关系性代价**（失眠、社交回避、关系消耗），**禁疾病预测/病程/概率**；② 单篇只显影 1 条轨迹，不做恐惧叠加；③ **必须给出口，且出口不能只有"来找我们"**；④ 禁"再不 X 就晚了"式时间胁迫；⑤ 触及自伤/自杀/幻觉 → 立即停止内容化
- **❌ 最恶毒也最会被写出来的那一版**：《不处理原生家庭创伤，你女儿会重复你的人生》——命运预言 + 恐惧胁迫 + 代际对立，三重红线。**看到它就打回。**
- **注意：「等我不忙了再说」不属于 P4**。它是"时机幻觉/启动惯性"，落点是一个动作而不是一次加压 → 它属于 **P11**。别把一个完全无害的高频场景，连坐进 P4 的高危护栏里。

#### P5 · 去羞耻化 ｜ 固定 1 ｜ ★★ 长期客
- **公式**：把"性格缺陷"重述为"当年的适应"（机制）＋ "这不是你的错，是你当时只能这样" ＋ 常见性（**不编造统计**）＋ 一句**许可**（你可以不再一个人扛）
- **✅ 例**：「你北大毕业、offer 不错、父母健在——所以你连'我不好'这三个字都说不出口。**条件好不是不痛的理由，只是不敢喊疼的理由。**」
- **❌ 写砸**：① **替读者断定他的童年**（"那是因为你小时候，先认错的人挨骂最少"）——我们对这个读者的童年一无所知，这是一次远程个案概念化；② 单因归罪原生家庭/父母（煽动代际对立 + 不可证）；③ 编造统计（"90% 的讨好型人格都有童年创伤"）；④ 把去羞耻写成免责（"你什么都不用改，是这个世界的错"）——爽了但取消了行动的必要性
- **✅ 合规改写（条件式，把断言权交还读者）**：「你总在道歉，连别人踩了你的脚你都说'对不起'。**那不一定是懦弱。有些人是在很早的时候学会的：先认错的人，麻烦最少。**（如果这对你来说有点耳熟——它曾经保护过你，只是现在不需要了。）」
- **⚠️ 最细的那条线**：**羞耻只能被命名并解除，绝不可被用作动力。** 判据：这句话让读者觉得"原来我可以不用扛"（✅），还是"我连这都做不到"（❌）？
- **⚠️ 中国语境的关键区分**：这里的羞耻**不是**"看心理咨询丢人"（西式病耻感），而是**"我这么好的条件还搞不定，是我不行"**（自我归责）。照搬西方反污名叙事不是最优杠杆——打穿"你不是不够坚强，是方法不对"才是。（"别人会不会知道 / 会不会留记录"那一半**不在 P5**，在 **P9 子形态 b**。）

#### P6 · 失效归因 ｜ 弹性 ≤1 ｜ 高认知换购型（★，证据待验证）
- **公式**：你试过的 X（具体到那件事）＋ 它为什么会失效（**机制，不是否定他的努力**）＋ 什么条件下它才可能有效 ＋ 那个条件里你自己造不出来的部分是什么
- **✅ 例**：「冥想 APP 你连打卡 90 天，可你依然在会议上突然大脑一片空白。**冥想能训练你不被念头带走，但它训练不了'被人盯着看'这件事对你意味着什么。**（如果你的卡点只是'脑子太吵'，冥想大概率就够了。如果它每次都发生在**有人看着你**的时候——那是另一回事。）」
- **❌ 写砸**：① 贬低读者试过的方法（"看书是没用的，只有咨询有用"）——傲慢 + 绝对化 + 变相疗效承诺；② 改写成"只有我们能解决"；③ 变成"这 5 个方法都是错的"式反面清单
- **护栏**：不贬低任何既有方法；不承诺"这次一定有用"；**必须诚实标出"什么情况自助就够了"**（不能把所有人都赶去付费）
- **⚠️ 证据诚实标注**：这条的"长期客 ★★"在调研稿里是**虚标**——唯一具名个案是**蒸发**的（预沟通后无人跟进、没付钱），投放评分 7.65 是**从未投过的纸面分**。且它的天然读者是"做过咨询且不满意"的人，**期待值与挑剔度高，二次脱落风险高于新客**。→ **降为观察级弹性位，不进"必配"。**

#### P7 · 价值论证 / 替代方案 ｜ 弹性 ≤1 ｜ **2026 年的头号缺口（Pull 承重墙）**
- **卡点**：「我知道我怎么了，也不觉得丢人——但我跟 AI 聊聊 / 跟闺蜜吐槽 / 自己看书好像也行，凭什么要花几百块 × 十几次？」
- **公式**：一个免费替代品**真实有效的部分**（诚实承认）＋ 它**结构上做不到的那一件事**（机制，不是贬低）＋ 那件事为什么需要一段**关系**而不是一个**信息源** ＋ **诚实的分界：什么情况下 AI / 朋友 / 书真的就够了**
- **✅ 例**：「你跟 AI 聊到凌晨两点，它一句都没打断你，也没评判你——**这确实有用，而且它可能比你身边任何一个人都更耐心。** 但有一件事它做不到：它不会在你第三次说'我没事'的时候停下来问'那你上次说没事的时候，也是这样吗'。（如果你需要的是把话说出来——AI 和朋友就够了。如果你每次说完都更空——那说明要处理的不是'说出来'。）」
- **❌ 写砸**：① 贬低替代品（"AI 只是人工智障"）；② 打价格战 / 促销；③ 不给诚实分界（把所有人都赶去付费）
- **互斥判据（和相邻三条切干净）**：**P4** = 他不觉得这个**问题**值得处理（问题侧）｜**P7** = 他觉得问题值得处理，但**不觉得付费专业服务是对的解法**（方案侧，**还没试**）｜**P6** = 他**试过了**，失败了（经验侧）｜**P8** = 他已经决定买专业服务，卡在**买哪种 / 找谁**
- **为什么是我们的优势位**：个人博主**写不了**这一篇——他们卖课/卖广告，写"什么时候 AI 就够了"等于自杀。**只有卖咨询、且敢说"有时候你不需要咨询"的机构，写这一篇才有可信度。**

#### P8 · 求助路径显影 ｜ 固定 1（取证前可退回 P2）｜ ★★☆ 高客单
- **范围（安全子集）**：怎么挑咨询师 / 自助 vs 专业的分界 / 咨询的设置与费用 / 换人不是你的失败 / 什么时候该换人
- **⛔ 不在范围内（→ F1 公域冻结）**：疗法对比（EMDR / 图式 / IFS / CBT 哪个适合你）、服药与咨询的先后顺序 —— **这是事实上的治疗建议，撞《精神卫生法》§23（心理咨询人员不得从事心理治疗或精神障碍的诊断、治疗）。这不是限流问题，是执业资质问题。**
- **公式**：一个已经确认自己议题的人的具体处境 ＋ 他面前的几个选项（各自适合谁、不适合谁）＋ 判断依据（**不是"选我们"，是"怎么选"**）＋ **什么情况自助就够 / 什么情况必须转介**
- **✅ 例**：「怎么判断一个咨询师适不适合你？**不是看头衔，是看前三次里有没有这三件事发生**——① 他有没有跟你确认过"我们要一起解决什么"；② 你有没有在某一刻感到被听懂了（哪怕只有三秒）；③ 你能不能说出"刚才那句话让我不舒服"。**三次之后一件都没发生，换人不是你的失败。**」
- **❌ 写砸**：① 写成自家服务广告（"我们有 XX，快来"）→ 硬广限流；② 疗法优劣排行（"EMDR 是最有效的创伤疗法"）→ 极限词 + 疗效承诺，双违规；③ **对读者的用药 / 停药 / 诊断给建议、评价医生的决定** → 红线中的红线；④ 退化成自助方法清单（见 G9）
- **护栏**：只做流程科普；**必须含"自助足够 vs 需要专业帮助"的分界**；自伤/自杀/双相/精神病性 → 只做识别与转介 12356

#### P9 · 首访祛魅 ｜ **固定 1（转化层打头阵）** ｜ 低风险 · 提全盘转化率
三个**子形态，一篇只取一个做落点**（不要三合一，那会让一篇有三个落点）：

| 子形态 | 卡点 | 公式 |
|---|---|---|
| **a · 过程时间线** | "第一次会发生什么？500 块值不值？" | 把"第一次"拆成可预期的时间线（进门 → 前十分钟 → 中间 → 结束）＋ 破除三个想象（不需要哭 / 不需要说得完整 / 不会被打分）＋ 一个真实但**脱敏且合成**的片段 |
| **b · 保密与记录** ★零风险高信任 | "会不会被家人/单位知道？会留档案吗？影响考公吗？" | 我们的保密制度是什么 ＋ 记录存在哪 ＋ **我们不是医疗机构、不做诊断、不产生精神科就诊记录** ＋ **保密例外有哪几条（诚实讲，不藏）** |
| **c · 表达门槛** | "我打开预约页，坐了二十分钟一个字没打出来" | **你不需要说得很惨，也不需要说得很清楚** ＋ 有人的第一句是"我不知道从哪说起"——那句话本身就是一个很好的开始 |
- **✅ 例（a）**：「第一次 50 分钟里到底发生什么：前 10 分钟是设置和保密，中间是你说、他问，最后 10 分钟会一起看'我们要不要继续、往哪走'。**它不是审判，更像一次不用交作业的门诊。**」
- **❌ 写砸**：① 资质炫耀（读者要的是**过程可预期**，不是你的头衔）；② 承诺疗效（"三次就能好"）；③ 促销（"首次体验价 / 名额有限"）；④ **卖陪伴与依赖**——"有一个人，每周三晚上八点，只为听我说话而存在" ❌ **这句极其有效，也极其可疑：它卖的是陪伴，不是咨询，而且它精准命中最不该被卖陪伴的孤独人群。**
- **✅ 必须的收尾（指向 agency，不指向依赖）**：「**咨询的目标不是让你永远有一个周三晚上八点，是让你有一天不再需要它。**」
- **护栏**：**b 只陈述"我们自己的"制度与法律事实，绝不对"考公政审会不会查到"下结论**（那是别人的行政程序，我们无权背书）；来访故事走**案例三律**（G5）
- **⚠️ 证据诚实标注**：它的 ★★★ 是**借来的**——那批"谈妥了却蒸发"的人，失败原因是**没人跟进**（承接问题），不是"不知道首访长什么样"。**内容替代不了 CRM：修跟进 SOP（预沟通后 24h 内必须有人推动）的优先级高于本路径的产量。**

#### P10 · 可得性破解 ｜ 弹性 ≤1（小红书只做试水）｜ ★★★ 最高 LTV，但**主战场在搜索场**
- **公式**：一个结构约束的具体处境（时差 / 母语 / 本地供给 / 排期）＋ 这个约束怎么被绕过（机制，不是广告）＋ **绕过之后的真实代价（诚实：线上有什么做不到）**
- **✅ 例**：「你在纽约做了两年英文咨询，进步很大——**可你始终没法用英文讲清楚，我妈那句'我都是为你好'到底是什么味道。**（诚实地说：线上做不了危机干预。）」
- **❌ 写砸**：① 写成"线上咨询更好"的服务广告；② 回避线上的真实局限；③ 打价格战（"比线下便宜一半"）
- **鉴别规则（唯一可证伪的那条，务必照做）**：**如果这个结构约束明天消失，他会立刻预约吗？** 答"是"→ P10；答"我再想想"→ **他卡在别处**（多半在 P5 / P7）。一个用"太贵/太远"来正常化自己不去的人，他的真卡点不在这里。
- **⚠️ 渠道错配警告（红队最有价值的发现之一）**：客户质量是真的（海外客长期组 22% vs 脱落组 0%；人数 9.1% 贡献 17.2% 净实收；LTV 1.9 倍），**但没有任何证据表明小红书的推荐流能触达他们**——"纽约的中文咨询"是**搜索行为**，不是刷推荐流的行为。**→ P10 的主阵地是官网 SEO/GEO 长尾词（`纽约 中文 心理咨询`、`留学生 心理咨询 时差`）**；小红书上先用 1 篇试水并看**搜索来源占比**，不要一上来就占 1/7 产能。（主阵地归属**待老板拍板**。）

#### P11 · 临门推动 ｜ 弹性 ≤1 ｜ 文献里预测力最强的一环
- **卡点**：「我知道我该去，我也知道找谁，但我就是一直没约。**等这个项目结束吧 / 等孩子上小学 / 等我攒够钱。**」
- **公式**：把"去咨询"从一个**人生决定**，切成一个 **5 分钟的动作** ＋ 一个**具体的时间锚** ＋ 拆除"最好的时机"幻觉
- **✅ 例**：「你不需要决定'要不要开始心理咨询'。你只需要决定——**这个周三晚上八点，你要不要跟一个人说五十分钟的话。**」「最好的时机不存在，只有'更贵的时机'；你要做的不是等，是把第一步切到小得不需要下决心。」
- **❌ 写砸**：**它极易滑成促销**——"限时""名额""仅剩""再不约就没档期了" ❌❌。**只做行为切分与时间锚，不做稀缺性与紧迫感。**
- **为什么它必须存在**：HBM 中国样本里**行动线索 β=0.323 是全模型最强的独立预测因子**（比症状本身、比感知严重性都强），而"感知严重性"根本不是独立预测因子。**一个什么都想明白、就差临门一脚的高质量读者，如果没有这一条，会被旧规则误判进 P4（最高危、最低质那条）。**

#### F1 · ⛔ 治疗方案科普（公域冻结区）
- **内容**：疗法对比（EMDR / 图式 / IFS / CBT）、服药与咨询并行、"你这个阶段该先做稳定化还是直接处理创伤记忆"
- **处置**：**小红书等公域一律冻结**。只落**官网自有阵地**，且必须：① 有资质的咨询师**实名署名**；② 引用权威来源，不自创判断；③ 通篇只做**中性词条科普**（"这些名词各自是什么意思、通常用于什么"），**绝不出现"你这个阶段该选哪个"**；④ 用药 / 停药 / 诊断 / 医生的决定一律**回医生**，一个字不评价。
- **状态：待法务/合规拍板。拍板前，冻结即默认。**

#### PX · 带他进门（关系人轴专属）
- **卡点**：「我知道他需要，但他不肯去 / 我该怎么跟他说 / 要不要我先去」
- **公式**：一个"我看着他难受、我无能为力"的具体处境 ＋ 你**试过的那些话为什么会让他关得更紧**（机制）＋ **一个你自己能做的动作**（先说你自己 / 先照顾好你自己 / 你可以先来一次）
- **✅ 例（老板那个旗舰例子的正确落法）**：「孩子在学校被欺负了，回家一个字不说，你问什么他都摇头。你翻遍了'怎么让孩子开口'的帖子，试了三种方法——**他把门关得更紧。** 他关掉的不是青春期，是那扇你以为一直开着的门。**今天可以先不问他：说一句你自己的事。**」
- **❌ 写砸**：「三招让孩子开口」「让发脾气的女朋友冷静下来的 5 个做法」= **他者解码 = 毙**；「你男友是不是 NPD 的 7 个信号」= **给不在场的人贴标签 = 毙**（比自诊更危险）
- **护栏**：落点必须是**读者自己的动作**；禁教读者"诊断"身边人；受苦者涉危机 → 只给 12356

### 六条边界裁决样例（新手对齐锚点 —— 规则要少而硬，样例要多）

| # | 选题 | 主路径 | 判定依据（看落点） |
|---|---|---|---|
| 1 | 「你妈又打电话来了。她说'我都是为你好'，你胃开始疼。你三年没回家过年，每年编不同的理由。**你觉得自己冷血。**」 | **P5** | 落点若是"你不冷血，那是你当年学会的自保"（**赦免**）→ P5；落点若是"胃疼可能是你身体在替你划界"（**解释机制**）→ P3。**看最后三行在做什么：解释还是赦免。**（本例给的落点是赦免 → P5）。⚠️ 胃疼必须挂躯体分流句（G2） |
| 2 | 「你在纽约做了两年英文咨询…讲到我妈只能说 she was strict。你想过换中文咨询师，但怕别人觉得你矫情，也怕重新讲一遍。」 | **P10** | 用反事实测试：**母语约束明天消失，她会立刻约吗？** 会 → P10。（"怕矫情""怕重讲"是次要路径 P5/P9，不是主路径） |
| 3 | 「你读完了《身体从未忘记》，知道自己是复杂性创伤，也知道该找做 EMDR 的人。可你打开预约页，看到'请描述你的困扰'，**坐了二十分钟，一个字没打出来。**」 | **P9（子形态 c）** | 落点是"你不需要说得清楚才能来" → P9c。（"读完了书"不是 P6——**P6 要求他为那个信念付出过行动且失败；读书是念头，不是行动**） |
| 4 | 「孩子在学校被欺负了，回家一个字不说，你试了三种方法，他把门关得更紧。」 | **PX**（audience = 身边人） | 读者 ≠ 受苦者 → 走关系人轴。落点必须是妈妈自己的动作（"今天先不问他，说一句你自己的事"）。落成"三招让孩子开口" = 他者解码 = 毙 |
| 5 | 「你今年第三次在工位上哭。你关注了心理咨询的号，收藏了两篇'怎么挑咨询师'。**但你一直没约——你告诉自己，等这个项目结束。**」 | **P11** | 她什么都知道，只差行动线索。**（旧规则会把她判进 P4，即最高危、最低质的那条——这正是新体系要修的错）** |
| 6 | 「同事只是随口说了句'这个方案再改改'，你却在工位上僵住了…这不是玻璃心，如果这种'一点小事就被巨大情绪淹没'反复发生…」 | **P2**（次 P5） | 落点是"给这个反复发生的处境一个名字 + 一句'不是你的错'"。⚠️ **若落点变成"对上 3 条你就是 CPTSD"，它当场退化为 P3 标签版 → 返工。** 见 `assets/example-xhs-note.md` |

**判不出来 = 选题没想清楚，打回。** 两条最常用的可观测判别：
- **P3 ↔ P6**：笔记里那个被否定的东西，**读者为它付出过行动吗？**（读过书、做过咨询、搬过家、打过 90 天卡 → **P6**；只是一个念头"我就是懒" → **P3**）
- **P3 ↔ P5**：**看最后三行在做什么——解释（机制）→ P3；赦免（许可）→ P5。**

### 选型指引（拿到长文的一个知识点，问这一串）

```
知识点在手
  ├─【第 0 问·止损】换成别的行业/人群也说得通吗？ → 说得通 = 还在现象层，回去挖场景（五问）
  ├─【第 1 问·定人】能在脑子里看见这个人吗？（不是"焦虑的人"，是"产后 6 个月、婆婆天天在家、
  │                 丈夫说你想太多的新手妈妈"）→ 不能 → 回去定人
  ├─【第 2 问·受众】读者就是受苦的那个人吗？ → 不是 → 走关系人轴（PX / 九闸门重新语义化）
  ├─【第 3 问·慢性度】这一幕反复发生吗？ → 单点急性 → 加计数器改写成慢性版，或换题
  └─【第 4 问·落点】读完这篇，读者完成的那一个动作是什么？ → 那就是主路径（≠ 遍历排除）
```

**知识点 → 默认主路径 映射表**（从 pillar 长文拆笔记时直接查）：

| 长文里的知识点类型 | 默认主路径 | ⚠️ 常见误配 |
|---|---|---|
| 定义 / 症状描述 / "什么是 X" | **P2 处境命名** | ❌ 别做成标签测评（"你是不是 X"）→ 自诊完成态，不产出购买 |
| 机制 / 成因 / 神经科学解释 | **P3 归因矫正**（机制向） | ❌ 别把机制包装成病名；❌ 别给第三人贴标签 |
| 长期影响 / 共病 / 功能损害 | ⛔ **P4 → 默认不做**，改写成 P2 慢性版 | ❌ 疾病进程预言 = YMYL 红线 |
| "这是适应，不是缺陷" / 创伤的保护性功能 | **P5 去羞耻化** | ❌ 别替读者断定他的童年；❌ 别写成"都是父母的错" |
| 常见误区 / 为什么自助不够 / 康复的非线性 | **P6 失效归因** | ❌ 别贬低读者试过的方法 |
| 自助练习 / 技巧清单 | **P7 或 P8 的受控形态**（强制边界声明，见 G9） | ❌ 无边界声明的方法清单 = 打回 |
| AI 陪伴 / 朋友倾诉 / 自助书的局限 / 咨询到底值不值 | **P7 价值论证** | ❌ 别贬低替代品；❌ 别打价格战 |
| 如何选咨询师 / 咨询设置 / 费用 / 什么时候换人 | **P8 求助路径显影** | ❌ 别写成疗法优劣排行；❌ 别对用药给建议 |
| 疗法介绍（EMDR/图式/IFS）、药物与咨询的关系 | ⛔ **F1 → 公域冻结**（只落官网） | ❌ 一个字都别在小红书写 |
| 第一次会发生什么 / 保密与记录 / 说不出口 | **P9 首访祛魅**（a/b/c 三选一） | ❌ 别写成资质炫耀或促销；❌ 别卖陪伴 |
| 线上咨询 / 母语咨询 / 异地 / 跨时区 | **P10 可得性破解**（主战场在搜索场） | ❌ 别打价格战；必须诚实标注线上做不到什么 |
| "什么时候开始都不晚" / 拖延求助 | **P11 临门推动** | ❌ 别滑成"限时/名额"促销 |
| 案例片段（脱敏） | **任何路径**（它是叙事视角，不是路径） | ❌ 别把案例当路径；必须过案例三律 |

> **如果一篇 pillar 的知识点全都只能映射到 P1–P3**：说明它是纯认知层长文。处置：从长文外**补转化层素材**（首访流程 / 保密制度 / 怎么挑咨询师 / 线上可及性），凑够转化层配额；或换一篇 pillar。**一整套全是认知层 = 又生产了一批只会自诊、不会预约的读者。**

### 一套 5–8 篇的配比（**v2 假设配比**，见下方回撤线）

**固定位 + 弹性位**（这样不会像旧表那样算术打架）：

| | 7 篇标准套 | 5 篇精简套 | 8 篇完整套 |
|---|---|---|---|
| **固定** P2 处境命名（慢性） | 2 | 2 | **3** |
| **固定** P5 去羞耻化 | 1 | 1 | 1 |
| **固定** P9 首访祛魅（转化层打头阵） | 1 | 1 | 1 |
| **固定** P8 求助路径显影 | 1 | — | 1 |
| **弹性**（P1 / P3 / P6 / P7 / P10 / P11 / PX 中选，同一路径不重复） | 2 | 1 | 2 |
| **禁** P4 代价显影 · F1 治疗方案 | 0 | 0 | 0 |
| **合计** | **7** | **5** | **8** |

**硬约束（违反即返工）**：
1. **转化层（P8/P9/P10/P11）≥1 篇，且必含 P9 首访祛魅**（三条转化层里它最低危、最不像广告，用它打头阵）；上限 3。
2. **任一路径 ≤2 篇**；**唯一例外 P2 可到 3**（它是唯一被验证的闭环，不该被限产）。
3. **P4 生产配额 = 0**（需单篇书面审批 + 咨询师签字才能破例）；**F1 不进公域**。
4. **P3 ≤1，且落点必须是"机制 + 下一步"**，不得是"确认标签"。**P1 ≤1，且必须带人群锚点 + 慢性度锚点。**
5. **护栏与配额按"篇内出现的任一路径元素"计，不按主路径计。** 一篇笔记只要含代价显影元素（哪怕标成次要路径），就必须过 P4 全套护栏并占用 P4 配额。**P3 / P4 不得作次要路径。**（否则运营可以合法地把 6 篇恐惧内容标成"主 P2 次 P4"，所有闸门显绿灯。）
6. **5–8 篇并排看，必须像 5–8 个不同的人在不同处境里求救**，不能像目录（第一章定义、第二章区别、第三章方法）。**换人群 / 换议题就够了，不必靠换路径实现多样性。**

**⚠️ 这套配比是假设，不是结论（必须让老板知道）**：

| 假设 | 内容 | 证伪方式 / 回撤线 |
|---|---|---|
| **H1 慢性度** | 钩到长期客的概率与**慢性度**正相关，与**痛感强度无关** | 每篇打标（主路径 / 慢性度 / 情绪杠杆），跟踪该批线索的**北极星 = 首月完成 ≥3 次会话率**。H1 不成立 → 慢性度闸门与整套星级失效，配额重定 |
| **H2 自诊终点** | 停在"确认标签"的笔记产出自诊完成态，不产出购买 | CPTSD 集群零转化已强支持；若 P3 机制向笔记的预约关联显著为正，可放宽 |
| **H3 方法清单** | 方法清单主动关闭求助链（"收藏了 = 我能自己搞定"） | **这是待验证假设，不是定律**（见 G9）。A/B：同一知识点，纯场景版 vs 方法清单+边界声明版，跟踪 30 天预约关联 |
| **H4 转化层蓝海** | "竞品不做转化层 = 蓝海"（也可能是"平台压制服务型内容 = 雷区"） | **取证未做**：36 条样本里**没有一个机构号**。先抓 3–5 个机构号近 30 条笔记的标题/点赞/收藏。**回撤线**：转化层笔记 30 天曝光 < P2 的 1/5 且无预约关联 → 转化层内容改投官网/搜索场，公域配额退回 P2 |

> **诚实代价（别装看不见）**：这套配比会让**涨粉变慢**——涨粉最猛的是 P1 金句和自助方法清单，而我们限额 P1、给方法清单套上边界声明。**这是有意的取舍：我们不卖广告不卖课，我们卖咨询。粉丝数不是资产，能走到预约的人才是。** 但也别自欺：**P8/P10 这类"说明书型"内容很可能天生属于搜索场（官网 SEO / 站内搜索）而不是推荐流**——**它们的 KPI 不该是爆款率**，而应是"从其他笔记走过来的人的转化率"。用爆款率考核它们，运营会拿惨淡数据推翻整个体系。

### MECE 自证摘要（诚实版）

**互斥**：单一维度 = **笔记落点所消解的那一道闸门**（可观测、写在文本里、可复核）。
- 判定不靠"读者卡在哪"（不可观测、一对多），也**不靠序号遍历**（读者会跳闸：一句"医生建议你配合心理咨询"就能让人从认知层直接跳到转化层）。
- **层内并列时的 tiebreak**：取**客户质量更高、合规风险更低**的那条（P5 > P3；P5/P6/P7 > P4）。**P4 只能作为"排除掉其它所有闸门后的剩余项"被判到——它是兜底类，不是竞争类。**
- 高频易混对已给可观测判据：P3↔P6（**是否为那个信念付出过行动**）、P3↔P5（**最后三行是解释还是赦免**）、P4↔P11（**加压 vs 切小第一步**）、P7↔P6（**还没试 vs 试过了**）、P9↔P10（**心理阻力 vs 结构约束：约束消失明天就约吗**）。
- ❌ **不再宣称"两人独立打标必然一致"**——那是假的。改为：**两人可以对着落点文本复核**；判不一致 → 说明落点没写清楚 → 返工（这本身就是有用的信号）。

**穷尽**：链路封闭（从"无感"到"下单"是一条状态机）+ **外部攻击制**。
- ❌ **不再宣称"48 条样本全部落位 = 穷尽性已证明"**——那是循环论证（"归不进的一律判不值得做"，反例按构造永不出现）。红队用 17 条**外部**选题攻击，13 条归不进，其中 12 条是**真缺口**（关系人 / 价值论证 / 隐私记录）——它们已被本版补上（PX / P7 / P9b）。
- **归不进的四类要分开看**（不是一列）：**① 定义域外**（不在心理求助链上）→ 框架的边界，不是胜利；**② 合规毙**（制造对立、争议题）→ 护栏的功劳；**③ 无增量毙**（空泛鸡汤）→ 增量闸门的功劳；**④ 战略不做**（无边界声明的方法清单 / 他者解码）→ **这一类基于 H3 假设，标注为待验证，不是定论**。
- **穷尽性不能自证，只能被外部样本反复攻击**：**每季度取 15 条站外新爆款重跑归类测试；归不进率 >10% → 触发框架修订。**

**定义域声明**：本体系只管**「从无感 → 首次预约」的获客链**。已付费者的"好转降频/转段"属于**留存链**（读者状态与目标完全不同）——**不要把留存内容硬塞进这些路径**。要做另建一条链。

### 场景深挖五问（写每篇之前逐题回答，答不上来就说明还没挖够）

1. **谁**——不是"焦虑的人"，是「产后 6 个月、婆婆天天在家、丈夫说你想太多的新手妈妈」。**具体到能在脑子里看见这个人**。
2. **什么时刻**——不是"经常"，是「凌晨三点」「周日下午四点」「家长会结束走出校门的那一刻」。
3. **正在发生什么冲突/代价**——她此刻具体损失了什么（睡不着、不敢开口、关系在冷掉、孩子在退缩）。
4. **这一幕反复到第几次了**（慢性度计数器）——「第十七个周日」「每个月都要经历一次」「从大学到现在，第四段」。**反复 → 写（长期客）；单点急性（"昨天他跟我提了分手"）→ 加计数器改写，或换题**（钩来的是买止痛的脱落客，且系统性钩入我们接不了的超范围个案）。
5. **她自己为什么解决不了**——她试过什么、**机制上为什么无效**、那个条件里她一个人造不出来的部分是什么。同时必须能回答反面：**什么情况下她自助其实就够了？**（诚实分界；答不出来的，多半是我们在夸大自己）。

> **⚠️ 旧版第 4 问「不管它会怎样（她真正害怕的那个后果）」已被替换掉**，替换成慢性度计数器。原因：那一问直接生产 **P4 代价显影**（本版生产配额 = 0），而**恐惧不是求助的独立预测因子**——中国样本里"感知严重性"根本不进回归模型，进模型的是**自我效能（β=0.279）与行动线索（β=0.323）**。**决定客户质量的是慢性度，不是痛感强度。**

> **五问 ≠ 路径。** 五问是**挖场景**（这一幕是什么），路径是**定落点**（这篇把读者往前推哪一格）。两个都要有：**五问答不全 = 场景没挖够；落点数不出唯一一个 = 选题没想清楚。**

### 从长文拆笔记时怎么用（第 1 步拆分规划的硬要求）

长文（pillar）是**知识组织**（按主题/机制/方法分 H2），笔记是**场景组织 + 落点组织**。所以拆分不是切段落，是**换一套坐标系重新分箱**：

1. **先按配比定路径**（"这套里第 4 篇是 P9 首访祛魅"）——**路径是输入，不是输出。**
2. 再按路径**去长文（及临床素材）里找能落地的场景与知识点**："这条路径最能救的是哪个具体场景里的哪个人？"（用五问写出来）
3. **一篇笔记 = 一个场景 ＋ 一个落点 ＋ 长文里能支撑这个落点的那部分知识**。同一知识点可服务多个场景，不必和 H2 一一对应。
4. 5–8 篇 = 5–8 个**互不重叠的具体场景**，且**落点互不相同**（两篇落点撞车 = 撞车，改一篇）。

> **注意方向**：**配比是生成器**（路径 → 找场景），**落点判定是归类器**（拿到一篇 → 判它属于哪条 / 是不是废稿 / 有没有撞车）。**别用归类器去分配配额**——那会让转化层永远凑不够（真实读者身上总是同时带着更靠前的闸门）。
> 检验：把 5–8 篇的选题并排看，如果读起来像"目录"（第一章定义、第二章区别、第三章方法）——**返工**。它们应该读起来像 5–8 个**不同的人在不同的处境里求救**。

#### 危机内容：结构性排除（不进选题池）

- **自伤 / 自杀 / 双相 / 精神病性 / 严重功能损害**：**不做钩子、不做转化、不写干预方法**，只做**识别与转介**（12356 全国统一心理援助热线）。我们**没有危机干预服务与胜任力**（已有依规主动终止的实体先例）。
- **评论区 / 私信危机响应 SOP**（分类学能挡住危机**话题**，挡不住危机**读者**——一篇 P2/P5 笔记下面一定会出现"我也想过算了"）：
  1. 固定话术（不评估、不诊断、不承诺，只转介）：「谢谢你告诉我们。这里没法提供即时的危机帮助——**如果你现在很难熬，请打 12356（全国心理援助热线，24 小时）**。你值得有人在场。」
  2. 单一响应人 + 响应时限 + 留痕；
  3. **危机转介话术与任何商业 CTA 严格分离**（不得在同一条回复里既转介危机又推销咨询）。

#### 恐惧杠杆的实证底线（最容易犯，所以放最后再说一遍）

> **吓唬没用，给出路才有用。**
> 中国样本（N=446）多元回归：**"感知严重性"不是独立预测因子**；独立预测因子只有**自我效能（β=0.279）** 与 **行动线索（β=0.323，最强）**。恐惧诉求 meta 说它有效——**但那是在"配了效能陈述"的前提下**；高威胁 + 低效能 → 读者去解决"恐惧"本身（否认、回避、划走、取关）。
> **判据（读者读完的感受）**：应该是「**被理解了，而且知道下一步能做什么**」，不是「**被吓到了，必须马上花钱**」。后者不仅违规，也留不住长期来访。
> **戳痛点促成交的唯一边界**：读完之后，读者**更有能动性**（他多了一个选项：可以来找我们，也可以先自己试）= 可以；读完之后他**觉得只有我们能救他**（他少了所有选项）= 不可以——**哪怕它完全合规、哪怕它转化率高**。

#### 成功指标（内容侧的北极星）

> **用「首月完成 ≥3 次会话率」，不用 LTV / 总次数。**
> 前者衡量"来访者进入了工作状态"（长期客 73% vs 快速脱落客 8%），后者衡量"留得久"——**两者在伦理上不等价**。一个把 LTV 设为最高判据的内容体系，会系统性偏好那些**制造依赖**的场景；而临床上的成功是**来访者不再需要你**。（此项**待老板拍板**，红队不越权。）

## 1.6 发布排版规范（手机端实际渲染）

> **为什么单列这一节**：小红书正文**不渲染 Markdown**——`publish_note.py` 发布前会把 `**加粗**` `*斜体*` `` `代码` `` 的强调符**剥掉**（不剥的话正文里会出现字面的 `*`、`` ` `` 号）。所以你在 md 里写的加粗**到了读者手机上一点效果都没有**。正文能用的排版手段只有四样：**换行 / 空行 / emoji / 全角符号**。这一节把"每段 2–4 行、配 1 个克制 emoji"那句粗规则展开成可执行的排版口径。
> **计数不受影响**：`count_xhs.py` 只数汉字，emoji 与全角符号不计入——**排版怎么调都不会撞字数闸**，放心断句、放心留白。

**心理科普的排版调性（先立总基调，再看细则）**：我们是**治愈系**，不是带货号。排版的目标是"让一个正在 emo 的人读得下去、觉得被温柔接住"，不是"用感叹号和 emoji 把人摁在屏幕上"。**克制 > 热闹**：宁可素净得像一封手写信，也不要花哨得像促销传单。凡是让笔记"看起来像营销号"的排版动作（满屏 emoji、彩色符号墙、花哨分隔线、每句一个感叹号），一律不做。

### A · 首行钩子排版（决定 3 秒内滑不滑走）

- **首行 = feed 预览里唯一露出的那一两行**，也是读者决定"要不要点开"的地方。**把第 1 步挖出的那个最具体的痛点场景放首行**，别放标签、别放"今天想跟大家聊聊…"这种预热。
- 首行**控制在一行到一行半**（手机窄屏每行约 20 字会折行，太长会被折成一坨、钩点被埋）。一个完整场景句放不下时，**在句中断一次行**，把最扎心的半句留给折行后的视觉落点。
- 首行末尾**至多 1 个 emoji** 点情绪（🫥/😮‍💨 一类"说不出的憋闷"），**不叠**。首行绝不放 🔥💥❗ 这类促销感符号。

### B · 分段与空行节奏（手机端每行约 20 字的断句意识）

- **一个自然段 1–3 短句、不超过 3 行**（对齐手机折行：3 行差不多是一屏里一口气读完不费力的量）。段落太长（4 行以上一大坨）在手机上就是"劝退墙"。
- **段与段之间空一行**（一个空行即可，别空两行以上留大洞）。空行是心理内容最重要的呼吸感来源——它给读者"停一下、消化一下"的余地。
- **强调靠"独立成行"，不靠加粗**（因为加粗会被剥掉、到手机上没效果）。想让哪句金句被记住，就**把它单独拎成一段、前后留空行**——视觉上的孤立本身就是最强的强调。范例：把"你真的不孤单。"从上一段末尾拆出来单独成行，比给它加粗有用得多。**独立成行的金句全篇 2–4 处为宜**（CTA、危机声明、G1 出口句这类结构性单句段不计）——处处独立就稀释成"每句都空行"的稀碎感（见反面清单）。
- 分点信息用 `·` 或数字序号起头，**每点一行**；点与点之间不空行（同组信息贴在一起才看得出是一组）。

### C · emoji 三用法 + 心理类密度区间

emoji 只在**服务语义**时出现，三种正当用途：

| 用法 | 怎么用 | 例 |
|---|---|---|
| **① 序号 / bullet** | 分点时代替枯燥的数字（可选，`·` 也行） | 🌱 先做一件小事 / 💬 想聊聊… |
| **② 段末情绪点缀** | 一段情绪落点处收 1 个，替代语气词 | 「不是你的错🤍」 |
| **③ 首行 / 首图钩子** | 首行末尾 1 个点情绪 | 「…回放那七个字🫥」 |

- **密度（心理类偏克制）：全文 3–6 个，平均每 2–3 段最多 1 个**。**绝不每行都挂 emoji**、绝不一处叠 2 个以上（😭😭😭 / ✨✨✨ 是营销号特征）。
- **选温和治愈的**：🤍🌱🫂🍃🕊️😮‍💨🫥 一类；**避开高唤起/夸张/带货感的**：🔥💥😱❗‼️💯🉐🎁 一律不用（它们制造焦虑或促销感，与治愈调性冲突，也踩"贩卖焦虑"的观感线）。
- **白名单外的 emoji 按黑名单特征判**：不制造焦虑感/促销感即可用；功能性指向（如 👇 指评论区）允许、计入总密度。
- **正误对比要软化**：❌→✅ 的"错误示范"式斩钉截铁与心理科普的非评判基调冲突（G4 不代读者作结论）——话术改为**「常见误解 → 更准确的理解」**，符号弱化（用「误/更准」文字标签或 🌫️→🌱），绝不给读者自己的应对方式贴"错"字。
- 危机声明那一段**不放 emoji**（严肃信息不点缀）。

### D · 分隔与装饰符白名单

- ✅ **可用**：换行、空行、`·`（分点）、`｜`（同行分隔如页脚）、`「」【】`（引出短语/栏目）、`（）`（补充说明，**替代破折号**）。
- ⚠️ **克制/避免**：`——` 破折号做解释性插入（两个长破折号在手机上视觉很重，且是常见的"AI 腔"信号）——**改为换行成段、用括号，或直接改逗号/句号断句**；语气性短破折号**全篇至多 1 处**（判例：「那不是你不够努力——真的。」这类收在句尾的顿挫允许；「A——也就是B——所以C」这类解释插入不允许）。
- ⛔ **禁用**：花哨分隔线（`━━━━` `▁▁▁` `✦•✦•✦`）、装饰性 bullet（`▪️➤✦❖`）、彩色特殊符号堆（`★☆♦︎❤︎`）、整行 emoji 分隔（`🌸🌸🌸`）——**这些是营销号最一眼的特征**，出现即显廉价，掩盖不了内容单薄反而暴露它。

### E · 结尾互动 + 标签行排版

- **CTA 单独成段**（「你也有过这样的瞬间吗？评论区抱抱」），别和上文干货挤在一段。
- **可见标签行放在最末、与正文之间空一行**，一行内 `#标签 #标签 #标签` 用**空格**隔开（别用逗号、别每个标签换行）。标签行不放 emoji。
- 标签行与 frontmatter `hashtags` 双写一致（见 §1「话题标签」）。

### F · 危机声明的排版位置（G6 单独成段）

- 危机声明 12356 **单独成一段**，**不与商业 CTA 同句同段**（G6 硬闸）——排版上就是"轻引导 CTA 一段、空行、危机声明另一段"。
- 位置：放在**结尾轻引导之后、可见标签行之前**；用一对括号包成一行（`（本文为心理科普…12356）`）与正文拉开层次，安静、不抢戏、不点缀 emoji。

### ⛔ 反面清单（出现任一即"营销号观感"，返工）

- 满屏 emoji：几乎每行末尾都挂、或一处叠 3 个以上。
- 花哨分隔线 / 装饰 bullet / 彩色符号墙（见 D 禁用项）。
- 全篇靠加粗强调——**加粗发布后被剥，等于没排版**；关键句该独立成行却埋在长段里。
- 一大坨没有空行的"文字墙"（4 行以上不换气），或反过来每句都空行、通篇稀碎。
- 感叹号连用 / 每句一个 `！`、`❗`——制造亢奋，与治愈调性冲突。
- 破折号套破折号做多重解释性插入（`——…——…`）。
- 正文中段夹标签（`#焦虑`）——标签只在文末那一行。
- 平台热梗堆砌（"谁懂啊""一整个…住""听劝""别划走""救命"）——流量语感会消解心理议题的严肃性：**不进标题**，正文至多偶发一处且不承载信息。

> **卡片图（轮播插画）的视觉排版是另一套体系**，不在本节——见 `references/illustration-spec.md`（品牌调色板、文字入图、版式）。本节只管**发布文案正文在手机上的文本排版**。

### before / after 对照（同一段文案：裸排 vs 优化排版）

**❌ 反例一 · 营销号式（满屏 emoji + 感叹号连用）**

```

**❌ 反例二 · "正常但错误"式（语句通顺，却靠加粗强调 + 解释性破折号 + 长段不空行——这是旧范例的真实失败模式，发布后加粗被剥、排版彻底失效）**

```

**✅ 优化排版（首行场景直入 + 短句 + 空行呼吸 + 金句独立成行 + 1 个治愈 emoji）**
