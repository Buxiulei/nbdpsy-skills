# 小红书笔记规格 + 合规红线（心理科普类）

> 依据：小红书平台规则、《广告法》§9（极限词）/§17（医疗用语）、《交易导流违规管理细则》（2025-03-12 生效）、全国统一心理援助热线 12356（2025-05-01 前并线）。下列"硬性要求"违反任一即不合格。

## 1. 笔记结构规格

### 标题（决定点击的第一道关）
- **硬限 20 字**，重点前置。结构常见为「关键词 + 钩子词」。
- 四类钩子任选其一：
  - **数字体**：「3 个迹象说明你可能…」「5 步自救」
  - **痛点体**：「总说自己想太多？」「为什么我总是 emo」
  - **悬念体**：「我后悔没早点知道这件事」
  - **身份标签**：「i 人必看」「留学生码住」
- 符号 `｜``【】` + 1–3 个 emoji 点缀，不堆砌。

### 正文 —— 三段式骨架（**本项目强制结构**）

> **为什么必须是三段式**：纯科普易涨粉、难转化（读者收藏完就走，不知道你能帮他）；业务硬广易转化、难曝光（平台限流 + 投流不过审 + 读者反感）。**「痛点场景开头 + 科普干货主体 + 结尾轻引导」是两难之间唯一的解**——它同时对齐小红书用户的真实决策链路（痛点搜索 → 案例对比 → 私信咨询 → 社群成交），也符合投流审核的通过规律（全文绝大部分是科普干货、只有结尾一句轻引导的笔记更容易过审；通篇引导或开头即广告的，限流与拒审风险最高）。

| 段 | 占比 | 写什么 | 硬要求 |
|---|---|---|---|
| **① 痛点场景开头** | 约 15–20% | 用一个**具体的生活场景**（不是抽象定义）让读者"对号入座"：某个时刻、某句话、某种身体反应。首句就是 3 秒钩子。 | 场景必须具体可感（"凌晨三点又醒了，脑子里循环白天那句话"），**禁止**用"你是否常常感到焦虑？"这种问卷腔 |
| **② 科普干货主体** | 约 65–75% | 这是笔记的**主体和价值来源**：解释机制、给判据、给可执行的方法。分点写，每点都有信息量。 | 干货必须能独立成立（**读者不点你主页也有收获**）；数字/结论可溯源到源长文；这一段**完全不谈自家服务** |
| **③ 结尾轻引导** | 约 10%，**1–2 句** | 先给互动 CTA，再**轻**提一句"还有专业支持这条路"。**轻 = 陈述事实、给出选项，不催不促不承诺**。 | 见下方「轻引导写法」；**禁止**任何硬广特征（限时/优惠/名额/立即/私信我领），**禁止**站外导流（微信/二维码/外链） |

**轻引导写法（结尾这 1–2 句怎么写）**：

- ✅ 安全句式（陈述 + 留门）：
  - 「如果这些方法试过还是走不出来，那不是你不够努力——有时候只是需要一个专业的人陪你一起看。」
  - 「自我调节能解决一部分，剩下的部分交给专业陪伴也不丢人。主页有更多这个主题的内容。」
  - 「觉得有用就收藏。想聊聊自己的情况，评论区或站内私信都可以。」
- ❌ 反面（硬广特征 / 违规）：
  - 「限时优惠，现在预约立减 200」（硬广 + 促销）
  - 「私信我领取免费测评名额」（诱导 + 名额）
  - 「加微信 xxx 免费咨询」（站外导流，直接封号）
  - 「我们能治好你的焦虑」（疗效承诺 + 医疗违禁）
- **品牌锚句每篇只出现一次**（见 §3），放在结尾轻引导里自然带过，不重复刷。

**其它硬性要求**：
- **硬限 1000 字**；本项目**默认 300 汉字**（用户指定，只数正文主体）。小红书爆款经验区间为 500–800 字（更厚更易被收藏）——如要提收藏率可调到 500–600，但默认遵守 300。三段式在 300 字下的典型配比：开头 50–60 字 / 主体 200–220 字 / 结尾 30–40 字。
- 主体内每段 2–4 行；分点用 `·` 或编号；每段配 1 个克制 emoji（治愈系不要满屏）。
- 口语化、第一人称、像朋友说话；**共情先于科普**（这是心理内容区别于普通种草文的关键）。
- **结尾 CTA**（在轻引导之前或之内）：提问引评论 / 「收藏不迷路」/ 「评论区抱抱」。

### 话题标签 #
- **硬限 10 个**，建议 **5–10 个**，三层各 2–4 个：
  - **关联**（精准品类/品牌词）：`#心理科普` `#CPTSD` `#情绪内耗`
  - **热度**（蹭当下热点话题）
  - **长尾**（细分 + 身份/地域）：`#留学生心理` `#童年创伤` `#上海心理咨询`
- **位置**：作为可见文本 `#标签1 #标签2 …` 附在「发布文案」正文末尾（小红书惯例标签贴文末），**同时**写进 post 文件 frontmatter 的 `hashtags` 备查。只放 frontmatter 渲染时看不见、也没法复制，等于没标签。

### 图文轮播
- **硬限 18 张**（2024 年由 9 张扩容到 18），**建议 6–9 张**（经验值，无官方最优数）。
- **比例 3:4 竖版，1080×1440px**——显示面积比横版约大 40%、点击率最高。全套统一尺寸，防止滑动时跳动。
- **首图 = 门面**：大标题 10–15 字、字号醒目；安全区距顶 ≥150px、距底 ≥200px、左右各 ≥60px（关键文字别贴边，避免被 UI 遮挡）。
- 每张当 1 页 PPT：1 个要点 = 1 个大标题 + 4–6 条短点（信息密度要够，别只放一句空泛标题）。

### 选题口吻（心理/情绪类爆款角度）
内核：**情绪背后是未被满足的需求**。三种最稳的体裁：
1. **第一人称共情体**——"如果你也常常…，这篇写给你"。
2. **干货清单体**——编号清单，天然适合收藏。
3. **自救指南体**——痛点 + 可执行的小步骤。
热门母题：人际关系、心理学效应、恋爱心理、缓解内耗/焦虑、自我觉察。

## 1.5 选题的生死线：**从「现象」深挖到「场景」**（本项目最高优先级规则）

> **为什么这条最重要**：停留在现象层的笔记，读者的反应是"学到了"（收藏 → 划走 → 忘记）；深挖到场景层的笔记，读者的反应是**"这说的就是我"**（对号入座 → 情绪被击中 → 才可能付费）。
> **付费不是被科普说服的，是被场景击中的。** 科普只提供"可信"，场景提供"关我什么事"——两者都有，转化才发生。

### 现象 vs 场景（一眼看懂差别）

| | ❌ 现象层（泛，无痛感） | ✅ 场景层（具体，有痛感） |
|---|---|---|
| 例 | 孩子内向不爱说话 | 孩子在学校被欺负了，回家也一个字不说——**你问什么他都摇头，你该怎么撬开这扇门？** |
| 例 | 孩子内向不爱说话 | **你以为他只是性格慢热**，可很多孩子在真正出事之前，给父母的信号也不过就是"最近不太爱说话" |
| 例 | 亲密关系中的沟通问题 | 你们已经三个月没吵架了——**但也三个月没聊过一句超过五分钟的话**，这种"和平"比吵架更让你心慌 |
| 例 | 职场焦虑 | 周日下午四点，你开始胃疼、心跳快、反复看手机——**明天要开周会，而你已经这样过了十七个周日** |
| 例 | 原生家庭创伤 | 你妈只是打来一个再普通不过的电话，你却在挂断后蹲在阳台上哭了半小时——**你自己都说不清为什么** |

**判据（一票否决）**：如果这个选题**换成任何别的行业/人群也说得通**，它就还停在现象层。"孩子内向不爱说话"可以是教育机构、口才班、儿童摄影都能写的开头；而"被欺负了回家一个字不说、你问什么他都摇头"只有心理这条路能接住。

### 两条深挖路径（把现象升级成场景）

**路径 A · 处境具象化**（把问题放进一个具体的、正在发生的困境里）

> 公式：**现象 ＋ 具体处境 ＋ 恶化后果 ＋ 当事人无解**
> 「孩子不爱说话」＋「在学校被欺负」＋「回家也不肯开口」＋「你问什么他都摇头，怎么办」

**路径 B · 认知反转**（打破一个想当然的解释，指向被忽略的真相）

> 公式：**你以为是 X（无害的解释）＋ 但其实可能是 Y（被忽略的真相）＋ 证据/信号**
> 「你以为只是性格慢热」＋「但很多孩子出事前的信号就是不爱说话」＋「这三个信号要当回事」

### 场景深挖五问（写每篇之前逐题回答，答不上来就说明还没挖够）

1. **谁**——不是"焦虑的人"，是「产后 6 个月、婆婆天天在家、丈夫说你想太多的新手妈妈」。**具体到能在脑子里看见这个人**。
2. **什么时刻**——不是"经常"，是「凌晨三点」「周日下午四点」「家长会结束走出校门的那一刻」。
3. **正在发生什么冲突/代价**——她此刻具体损失了什么（睡不着、不敢开口、关系在冷掉、孩子在退缩）。
4. **不管它会怎样**——她真正害怕的那个后果（不是抽象的"变严重"，是"错过信号""孩子彻底不跟我说话了"）。
5. **她自己为什么解决不了**——她已经试过什么、为什么无效（"我问了他不说""我讲道理他更沉默"）。这一问最关键：**它是"你需要专业帮助"这句话唯一诚实的理由**。

### 从长文拆笔记时怎么用（第 1 步拆分规划的硬要求）

长文（pillar）是**知识组织**（按主题/机制/方法分 H2），笔记是**场景组织**。所以拆分不是切段落，是**换一套坐标系重新分箱**：

1. 先列出长文里的**知识点/机制/方法**（原料）。
2. 再问：**"这个知识点最能救的是哪个具体场景里的哪个人？"**——把答案写成一个具体场景（用上面五问）。
3. **一篇笔记 = 一个场景 ＋ 长文里能解答这个场景的那部分知识**。同一个知识点可以服务多个场景，同一个场景也可以调用长文多处知识——**不必和 H2 一一对应**。
4. 5–8 篇笔记 = 5–8 个**互不重叠的具体场景**（不是 5–8 个知识角度）。

> 检验：把 5–8 篇的选题并排看，如果它们读起来像"目录"（第一章讲定义、第二章讲区别、第三章讲方法）——**说明还是在按知识组织，返工**。它们应该读起来像 5–8 个**不同的人在不同的处境里求救**。

### ⚠️ 合规护栏：深挖场景 ≠ 制造恐慌（心理科普 YMYL 红线，违反即不合格）

路径 B（认知反转/风险预警）转化力最强，也**最容易滑向恐吓营销**。心理健康是 YMYL 领域，我们是心理咨询工作室不是医疗机构，以下红线必须守住：

- ✅ 可以写"**这些信号值得你当回事**"；❌ 不可以写"**不管就会出大事/后果不堪设想**"（制造恐慌、暗示严重后果）。
- ✅ 风险提示必须**有据**（源长文里能找到出处的研究/共识）；❌ 不可以为了钩子编造或夸大因果（"不爱说话 → 抑郁症"这种跳跃式吓唬）。
- ✅ 每篇给**建设性出路**（具体能做的第一步）；❌ 不可以只制造焦虑、把出路完全锁在"来找我们咨询"。
- ✅ 涉及自伤/自杀/重度精神障碍等高危议题时，**只做识别与转介引导**（12356 热线 + 建议就医），绝不暗示我们能处理危机——我们**没有危机干预服务与胜任力**。
- 判据：读者看完的感受应该是「**被理解了，而且知道下一步能做什么**」，不是「**被吓到了，必须马上花钱**」。后者不仅违规，也留不住长期来访。

## 2. 合规红线

| 类别 | 红线 | 后果 |
|------|------|------|
| **极限词**（广告法 §9） | 最 / 第一 / 唯一 / 独家 / 国家级 / 顶级 / 首选 / 根治 / 永久 / 100% / 彻底 / 绝对 | 最高罚 100 万；指向"本经营者"风险最高 |
| **医疗用语**（广告法 §17） | 治疗 / 治愈 / 诊断 / 特效 / 疗效 / 抗抑郁 / 药到病除（非医疗机构禁涉治疗） | 违规下架 |
| **站外导流**（导流细则 2025-03-12） | 联系方式 / 他平台账号 / 外链 / 口令 / 二维码 / 微信号 / 加群（笔记、评论、简介、私信、置顶、**图中**均算） | 限流 → 封号 |
| **硬广特征**（投流审核 + 平台限流） | 促销词：限时 / 优惠 / 特价 / 立减 / 折扣 / 免单；催促词：立即 / 马上 / 抓紧 / 仅剩 / 名额；诱导词：私信我领 / 扫码领 / 免费领取 + 名额 | 投流拒审、自然流限流 |

**危机声明惯例**（每篇在位，放正文末或末页）：

> 本文为心理科普，不构成诊断或治疗；如持续困扰（如情绪低落 ≥2 周、焦虑 ≥1 个月并影响生活），请寻求专业帮助。全国统一心理援助热线 **12356**（热线不替代系统咨询）。

### 禁用词 → 安全替换对照表

| 禁用词 | 安全替换 |
|--------|----------|
| 最有效 / 最好的方法 | 一种可以尝试的方法 |
| 第一 / 唯一 / 独家秘籍 | 分享个人整理 / 实用思路 |
| 100% 有效 / 彻底解决 / 根治 | 可能有帮助 / 有人反馈有改善 |
| 国家级 / 顶级 / 权威认证 | 参考公开资料 / 常见共识 |
| 治疗 / 治愈焦虑抑郁 | 缓解情绪 / 自我调节 / 情绪梳理 |
| 诊断你是不是抑郁症 | 自我觉察 / 自查参考（非诊断） |
| 抗抑郁 / 特效 / 药到病除 | 自助练习 / 减压方法 / 陪伴支持 |
| 加微信 / 扫码进群 / 留电话 | 评论区交流 / 站内私信沟通 |

> 注：`治疗 / 诊断` 等词在转述学术文献时本身合法（如"创伤治疗研究""认知行为治疗 CBT"），但在面向大众的小红书短文里，**用于描述本工作室服务或对读者作承诺**时一律替换。脚本只扫高置信违禁词，这类语境词靠本表语义自检。
