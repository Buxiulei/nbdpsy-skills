# 配图审查清单（轮播插图，逐张看图审）

> 审查对象：一篇笔记的成品配图目录（P01..PN）。
> 立场：预设有问题，逐张找证据；不确定按 FAIL。
> **铁律：判断性条目必须逐张打开图片查看**——不许只看文件名和尺寸就下结论；未打开图片得出的结论视为审查无效。

## 确定性检查（先跑，JSON 输出与 exit code 原样记入报告）

| 命令 | 预期 | exit 语义 |
|------|------|-----------|
| `python3 {SKILL_DIR}/scripts/check_images.py --dir <images> --pages <N>` | 页数齐全 + 尺寸合规 | JSON 输出；缺页或尺寸不合规 exit 1（尺寸判定：宽高比 3:4 容差 ±2%，最短边 ≥1080） |

`{SKILL_DIR}` = nbdpsy-content-reviewer 本 skill 目录；`<N>` = 该篇笔记规划的页数（对照 post-*.md 的 P 页标题数）。

## 判断性检查（逐张打开图片，逐条 PASS/FAIL + 证据 + 页码）

1. **图内中文与该页「页面文字」逐字一致**：对照 post-*.md 每页的"页面文字"，逐字核对图上渲染出的中文——无错字/漏字/多字/乱码/繁简混杂。AI 出图最常见翻车点就是错字，小字和专业缩写（CPTSD、ICD-11）重点盯。任何一个字不对 = 该页 FAIL，报告里写明页码 + 错在哪个字。
2. **莫兰迪基调无突兀撞色**：整套图以品牌三色 **#A8B5C4（雾霾蓝灰）/ #E8D8C4（暖米白）/ #C9D6CE（鼠尾草绿）** 系为主，文字色为深灰蓝 **#5A6B7B**；出现高饱和红/亮黄/荧光色等突兀撞色，或某页明显偏离整套色调 = FAIL。
3. **人物与人物卡一致且跨页同一人**：出现人物的页，形象须符合人物卡「**圆脸、齐肩微卷短发的东亚年轻女性，穿燕麦色针织衫**」，且 P1→PN 看起来是同一个人（发型/脸型/衣着不跳变）。
4. **构图完整无截断/畸变**：主体不被边缘切掉，人物无畸变手指、多余肢体、崩坏五官；文字块不压主体、不出血到边缘外。
5. **无二维码/联系方式/外链水印**：任何页出现二维码、微信号、电话、网址水印、第三方平台 logo = 直接 FAIL（站外导流是限流封号红线）。
6. **首图（P1）单独看有封面吸引力**：把 P1 当作信息流里孤立的一张图来看——大标题是否一眼可读、钩子是否成立、视觉是否干净有点击欲。P1 是内容页版式（信息密集小字）而非钩子封面 = FAIL。
