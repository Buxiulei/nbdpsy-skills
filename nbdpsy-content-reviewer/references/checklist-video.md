# 视频审查清单（短视频成片）

> 审查对象：nbdpsy-text-to-video 工作目录（shot-NN.mp4 / narr-NN.mp3 / narr-NN.mp3.cues.json / manifest.json / final.mp4）。
> 立场：预设有问题，逐条找证据；不确定按 FAIL。
> **铁律：判断性条目必须抽帧/抽段实际观看**——至少覆盖首镜、中间一镜、末镜三处（用 ffmpeg 抽帧或直接播放片段取证）；未看片得出的结论视为审查无效。

## 确定性检查（先跑，JSON 输出与 exit code 原样记入报告）

| 命令 | 预期 | exit 语义 |
|------|------|-----------|
| `python3 {SKILL_DIR}/scripts/check_video.py --workdir <dir>` | 时长/时间轴/成片完整全过 | JSON 输出；任一未过 exit 1。检查项：每镜 shot-NN.mp4 ffprobe 实测时长 ∈[4,15]±0.5s；cues 每镜单调递增且末句 ≤ 旁白实测时长+0.2s；final.mp4 总时长 ≈ Σ每镜时长 ±10%；总时长 ∈[--total-min,--total-max]（默认 30–180s，可按用户对该条视频的要求覆写） |

`{SKILL_DIR}` = nbdpsy-content-reviewer 本 skill 目录。

## 判断性检查（抽帧/抽段观看，逐条 PASS/FAIL + 证据 + 时间点）

1. **字幕与旁白同步**：在 final.mp4 上抽 3 处句边界（首镜一处、中段一处、末镜一处），实听实看：字幕切换点与旁白该句起止的偏差 <0.5s。任一处 ≥0.5s = FAIL，写明时间点与实测偏差。
2. **旁白完整无截断、无异常慢放画面**：逐镜听旁白结尾是否被切字；看画面是否有为凑旁白时长而出现的异常慢放/定格拖时间。发现即 FAIL，写明镜号。
3. **各镜风格/人物一致**：抽看的三镜（以及任何可疑镜）画风、色调、人物形象一致，不出现"换了个人/换了个画风"的跳变。
4. **画面内无任何文字（字幕除外）**：AI 生成画面里渲染出的伪文字/乱码招牌/图内标题都算违规——产线规则是文字一律走字幕层，画面本身必须无字。发现即 FAIL，写明镜号与时间点。
5. **片尾危机声明在**：结尾以字幕或画面呈现心理援助信息，必须含 **12356**（全国统一心理援助热线）。抽看末镜取证；缺失 = FAIL。
6. **BGM 不盖旁白**：抽听的三处旁白清晰可辨，BGM 明显低于旁白（产线基准：比旁白低约 12dB 相对响度）；任一处旁白被 BGM 淹没 = FAIL。
7. **AI 标识状态在交付说明中声明**：交付说明（或报告本条）必须写明成片是否已叠加"AI 生成"标识（manifest 的 `ai_label` 是否生效、画面上是否可见）。叠加与否由发布者决定，但**状态必须如实声明**——未声明 = FAIL。
