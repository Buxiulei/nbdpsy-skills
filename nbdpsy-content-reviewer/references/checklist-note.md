# 笔记审查清单（小红书图文，逐篇审）

> 审查对象：nbdpsy-xiaohongshu-creator 产出的 post-*.md（一篇一审，一套笔记逐篇出结论，任一篇 FAIL 则该篇返工）。
> 立场：预设有问题，逐条找证据；不确定按 FAIL。

## 确定性检查（先跑，JSON 输出与 exit code 原样记入报告）

路径说明：`../nbdpsy-xiaohongshu-creator/` 指安装后与 nbdpsy-content-reviewer 同级的兄弟 skill 目录；命令在本 skill 目录下执行，或换算为绝对路径。

| 命令 | 预期 | exit 语义 |
|------|------|-----------|
| `python3 ../nbdpsy-xiaohongshu-creator/scripts/count_xhs.py <post>` | 正文字数与页数达标 | JSON 输出；区间外 exit 2 |
| `python3 ../nbdpsy-xiaohongshu-creator/scripts/check_compliance.py <post>` | 无违禁词命中 + 危机声明在位 | JSON 输出；任一命中/缺失 exit 1 |

（两脚本内部口径以其自身实现为准，本清单只约定命令与 exit 语义，不复述内部规则。）

## 判断性检查（逐条 PASS/FAIL + 证据 + 位置）

0. **【生死线·先审这条】选题是「具体场景」而不是「泛现象」；主路径唯一且标注正确；正文三段式在位**。这条不过，后面做得再漂亮也是白做——读者的反应会停在"学到了"（收藏→划走），而不是"这说的就是我"。六个子判据：
   - **① 一票否决（换行业测试）**：把这个选题换成别的行业/人群还说得通吗？「孩子内向不爱说话」教育机构、口才班、儿童摄影都能写 = 还在现象层 = **FAIL**；「孩子在学校被欺负了回家一个字不说、你试了三种方法他把门关得更紧」只有心理这条路能接住 = PASS。
   - **② 场景要素在位**：正文开头能读出**谁**（具体到能想象出这个人）、**什么时刻**（凌晨三点/周日下午四点）、**正在发生什么冲突或代价**；抽象问卷腔开头（"你是否常常感到焦虑？"）= FAIL。标题应承载场景而非知识点命名（「CPTSD 和 PTSD 的区别」= FAIL；「一句'方案再改改'，我在工位上僵住了」= PASS）。**慢性度**：单点急性（"昨天他跟我提了分手"）且未加计数器改写 = FAIL（钩来的是痛感一降就走的止痛客）。
   - **③ 主路径唯一且标注正确**（**新增，按 xiaohongshu-spec §1.5 的 12 条路径审**）：frontmatter 有 `scene_path`（唯一一条）。**审查方法：只看落点——正文最后三行（不含危机声明与标签）在让读者完成哪一个内在动作？**（认领痛感 / 命名处境 / 换掉归因 / 卸下自责 / 明白为何试过没用 / 想清楚凭什么花钱 / 知道怎么选 / 预演首访 / 破解约不到 / 今晚就把它约掉）。
     - 落点数不出唯一一个，或落点与标注的 `scene_path` **对不上** = **FAIL**（标注造假比不标注更坏）。
     - `scene_path` 写成两条（"P2+P4"）= FAIL。次要路径 >1 条，或次要路径填了 **P3 / P4** = FAIL。
     - 两条最常用判别：**P3↔P6** 看"读者是否为那个信念**付出过行动**"（读过书/做过咨询 → P6；只是念头"我就是懒" → P3）；**P3↔P5** 看"最后三行是**解释**（机制）还是**赦免**（许可）"。
   - **④ 是否落入红队点名的高风险 / 低质量路径**（任一命中 = **FAIL**）：
     - **P4 代价显影（生产配额 = 0）**：出现"不管它会怎样"式恶化轨迹、疾病进程预言（"三年后会发展成抑郁症""你女儿会重复你的人生"）、时间胁迫（"再不 X 就晚了"）= FAIL。**注意按"篇内出现的任一路径元素"审，不按主路径审**——一篇标着 `scene_path: P2` 但正文在做恐惧显影，照样 FAIL（这正是配额可被规避的那个洞）。
     - **P3 标签版**：落点是"确认标签"（"对上 3 条你就是 CPTSD""你是不是讨好型人格"式自测清单）= FAIL（自诊完成态，零转化）。术语（CPTSD/回避型依恋）进标题或做落点 = FAIL；正文里超过一次的非解释性使用 = FAIL。
     - **他者贴标签（G3）**：给读者的父母/伴侣/子女/上司下人格或诊断判断（NPD/PUA/回避型/自恋）= FAIL。
     - **他者解码**：落点是"如何让对方改变"（"三招让孩子开口""让发脾气的女朋友冷静的 5 个做法"）= FAIL。身边人（`audience: 身边人`）笔记的落点必须是**读者自己能做的动作**。
     - **F1 公域冻结**：出现疗法对比（EMDR/图式/IFS 哪个适合你）、服药与咨询的先后顺序、对用药/停药/诊断的建议 = **FAIL（法律级，不是限流级）**。
     - **卖依赖（G10）**：收尾指向依赖（"有一个人只为听你说话而存在"）而非 agency（"有一天你不再需要它"）= FAIL。
     - **无边界声明的方法清单（G9）**：纯技巧清单且没有"这些方法在 X 情况下不够用 → 那时候需要 Y"的边界声明 = FAIL。
   - **⑤ 三段式 + 全局硬闸在位**：痛点场景开头（15–20%）→ 科普干货主体（65–75%，**完全不谈自家服务**、能独立成立）→ 结尾轻引导（1–2 句，不催不促不承诺）。主体段夹带服务推销、或结尾出现硬广特征（限时/优惠/名额/立即预约/私信我领）= FAIL。逐条查硬闸：**G1 最低出口**（正文末有一个今天能做、门槛低于付费的小动作；只共情不给出口 = FAIL）、**G2 躯体分流**（写到胃疼/失眠/心悸等躯体症状却没有"先排除器质性原因"的分流句 = FAIL）、**G4 不代读者作结论**（"那是因为你小时候…"式断言 = FAIL，须条件式）、**G5 案例三律**（来访故事无"综合多人改写"声明，或城市+职业+年龄+特殊经历 ≥3 项可反向识别 = FAIL）、**G6 危机与 CTA 分离**（12356 与商业 CTA 同段 = FAIL）、**G7 CTA 黑名单**（私信我/扣 1/加我/限时/名额 = FAIL）。
   - ⚠️ **YMYL 总判据**：深挖场景 ≠ 制造恐慌。"这些信号值得当回事" = OK；"不管就会出大事/后果不堪设想" = **FAIL**。风险提示必须在源长文有出处，不得为钩子夸大因果（"不爱说话→抑郁症"式跳跃 = FAIL）；涉自伤/自杀/重度精神障碍只做识别与转介（12356 + 建议就医），任何"我们能处理危机"的暗示 = FAIL。读者应感到「被理解了，且知道下一步能做什么」，不是「被吓到了，必须马上花钱」。
   - **⑥ 多篇一起审（配比是否失衡）**：5–8 篇并排读——
     - 像一份目录（第一章定义/第二章区别/第三章方法）= **整组 FAIL**（还在按知识组织）；应像 5–8 个不同的人在不同处境里求救。
     - **两篇落点撞车**（`scene_path` 相同且落点动作相同）= 整组 FAIL，改一篇。
     - **配比失衡（整组 FAIL）**：转化层（P8/P9/P10/P11）**< 1 篇，或缺 P9 首访祛魅**；任一路径 > 2 篇（**P2 例外，可到 3**）；**含 P4 元素的笔记 ≥ 1 篇**（未走书面审批）；P3 > 1 篇；P1 > 1 篇或 P1 没带人群/慢性度锚点。

1. **每个数字/事实能在源 pillar 原文找到**。对照 fetch 下来的源长文 drafts 文件逐个数字反查：笔记中出现的每处统计、每条引语、每个专有结论，都要在源文中找到原句或等价口径。源文里没有的"新事实" = FAIL（哪怕它碰巧是真的——产线铁律是不新增未验证事实）。
2. **每页提示词含品牌调色板与人物卡，且图内文字已写入提示词**。逐页核对：提示词含品牌主色 **#A8B5C4 / #E8D8C4 / #C9D6CE**（强调赭红 #A34B3A / 标题深色 #2B3A4A 是**可选**设计工具，用不用都不影响达标，别拿它当 FAIL 理由）；出现人物的页含人物卡「**圆脸、齐肩微卷短发的东亚年轻女性，穿燕麦色针织衫**」；该页"页面文字"里要显示在图上的中文，已用引号逐字写进提示词。缺主色/人物卡/文字入图任一项 = 该页 FAIL。
3. **标签双写**：frontmatter `hashtags` 与文案末尾可见 `#标签1 #标签2 …` 行**同时存在且一致**。只有 frontmatter 没有可见标签行（或反之）= FAIL。
4. **危机声明 12356 在文案内**（发布文案单元里可见，不能只在配图提示词里）。⚠️ **仅适用于心理科普笔记（拆长文）；咨询师推介笔记不判此条**——推介是人物介绍非科普内容，按 `../nbdpsy-xiaohongshu-creator/references/counselor-note-spec.md` §3 不含 12356 声明（其合规校验走 `check_compliance.py --no-crisis`），此篇缺 12356 不判 FAIL；其余判据（站外导流、极限词、医疗词、事实可溯源、排版等）照审。判断是否推介笔记：frontmatter/内容取材自某位咨询师公开资料、卖点是"这个人可信"而非科普干货。
5. **钩子首句 ≤20 字且有停留价值**：数字数；再判断它是否用痛点/场景/数字抓人——平铺直叙的说明句（"本文介绍 CPTSD 的成因"）即使 ≤20 字也是 FAIL。
6. **内容页信息密度充足（密度 = 图文共同表达的信息量，不是字数）**：中间内容页（非封面/末页）的干货密度来自**图文双通道**——把图和字**一起读**，能否获得一个"信息完整、对得上自己"的认知？逐页核验两条硬指标（对照 illustration-spec 第 3 节「内容页核心」①）：① **每条信息/每格/每节点都配一个各不相同、把该点画成具体场景的手绘小图或简笔小人**（贴合含义、非装饰圆点；图在叙事不在装饰、抽象点要落成具体画面如日历/曲线/并列症状小人）；② **满版、分区清楚、一页约 6–10 个信息点**。版式可为 17 种之一（分区清单/对比表/流程/象限/隐喻/数据/拆解/因果链/时间轴周期/自查清单/恶性循环圈/情景对话/身体地图/误区纠错/换个想法/程度光谱/概念定义卡）；**一套 6–9 页应混用 3–5 种、别整本一个版式**。少格版式（象限/对比表/身体地图等）应有"格内二次加密"（每格内嵌具体情境小图+子标注），只剩几个大标签 = FAIL。**FAIL 判据**：只有几行纯文字列表、图标是装饰圆点、没有分区、一页只有 1–2 条、或"**一个大隐喻 + 几个抽象标签**"（图在装饰、文字在贴标签、合起来没讲清一件事，尤以隐喻图/拆解图易犯）。**粗黑标题、赭红高亮等是可选设计项，用不用都不算问题，别拿它当 FAIL 理由。** 文字总量约 80–120 字为宜，但不为凑字数堆空话。
7. **页面文字连成故事线无跳跃**：把 P1→PN 的"页面文字"顺序连读一遍，检查封面钩子→内容页展开→末页行动建议的叙事是否连贯，页与页之间有无断层、重复或自相矛盾。
8. **无任何站外导流**：微信/微信号/二维码/加群/网盘/公众号/电话等一律不得出现（含正文、标签、每页提示词与页面文字）。脚本扫的是高置信词表，这里做语义兜底——变体暗示（"想聊的看我主页简介"）也算 FAIL。
9. **发布文案排版按 §1.6 规范（手机端实际渲染，非营销号观感）**：口径见 `../nbdpsy-xiaohongshu-creator/references/xiaohongshu-spec.md` §1.6「发布排版规范」。逐条看「## 发布文案」区块的**文本排版**（小红书正文不渲染 Markdown，加粗发布后被剥，看的是换行/空行/emoji/全角符号的实际效果）：
   - **无排版 / 整坨文字墙 = FAIL**：全篇挤成一两大段、几乎没有空行呼吸、4 行以上不换气的段落——手机上就是劝退墙。**反向稀碎同 FAIL**：超过 2/3 的自然段是单句段、且这些段不是 CTA/危机声明/G1 出口句等结构性段落（金句独立成行全篇 2–4 处为宜，处处独立=强调稀释为零）。
   - **首行不是钩子 = FAIL**：首行放了标签、"今天想跟大家聊聊…"式预热或空泛说明句，而非第 1 步挖出的具体痛点场景。
   - **emoji 超密度 / 营销号式花哨 = FAIL**：全文 emoji 明显超过 6 个、几乎每行都挂、或一处叠 ≥3 个（😭😭😭/✨✨✨）；用了高唤起/带货款 emoji（🔥💥😱❗‼️💯🎁）；出现花哨分隔线（`━━━`/`✦•✦`）、装饰性 bullet（`▪️➤❖`）、彩色符号墙、整行 emoji 分隔、感叹号连用/每句一个 `！`。心理类调性偏克制，这些一律判 FAIL。
   - **强调错位 = FAIL**：通篇靠 `**加粗**` 强调（发布后被剥＝等于没排版），关键金句该独立成行却埋在长段里。
   - **危机声明/标签排版 = FAIL**：危机声明未单独成段（与 CTA 同段，已由判据 0⑤ 的 G6 覆盖，此处只看排版落地）；可见标签行没有和正文空行隔开、或标签夹在正文中段、或每个标签换行/用逗号分隔。
   - ⚠️ 边界：排版本身不数汉字（emoji/符号不计），**别拿"用没用赭红/粗黑"当 FAIL 理由**（那是配图的可选设计项，与正文文本排版无关）；判据只针对"读者手机上看到的文本观感"。
